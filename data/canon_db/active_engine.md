## 2. 世界觀核心設定

### 2.1 世界性質

《武裝學院的二三事》的世界是一座被靈力科技滲入日常毛細血管的現代都市世界。
通行牌、感應柱、靈軌、公告系統、醫療監測、資料板、結界設備、紙本封存、生活靈紋與校園門禁，像路燈、電車、校園白板一樣自然存在於生活中。

這些設備服務生活，也承接制度壓力；它們可以帶來便利，也可能成為外部勢力鑽入縫隙的工具。

#### 2.2.1 武裝成長、戰力階段、生命本質與壽命尺度

異能武裝不是外掛兵器，而是使用者靈魂與能力的伴生具現媒介；能力本源始終屬於本人。本人與武裝之間的成長、理解與承載能力，會共同影響戰力、生命本質與壽命尺度。

#### 五階戰力與生命階段

本作正式能力階段依序為：

**初顯者 → 成形者 → 契合者 → 成熟者 → 超脫者**

| 階段 | 核心定義 | 能力與戰力位置 | 生命尺度裁定 |
|---|---|---|---|
| 初顯者 | 能具現異能武裝並開始使用自身能力者。 | 能使用基礎效果；穩定度、精度、判斷與代價理解仍有限。 | 一般生命尺度。 |
| 成形者 | 已建立穩定能力用法與自身基本打法者。 | 可重複運用招式與戰術，在訓練／競技中形成清楚優勢與限制。 | 一般生命尺度。 |
| 契合者 | 本人與異能武裝已高度契合者。 | 能依場地、對手、隊友與任務改變能力用法；已是成熟前的高水準戰力。 | 尚未因階段本身取得長壽；不得誤寫為成熟者。 |
| 成熟者 | 已掌握自身能力本質規則，且本人、武裝與生命本質完成共同進化者。 | 不只使用效果，而能穩定運用自身能力為何成立、能作用於什麼與必須承擔什麼代價。 | **凡成熟者必然進入顯著長壽尺度。** |
| 超脫者 | 已掌握自身能力相對應之世界法則的極少數存在。 | 能在同源能力範圍內觸及世界法則層級；不是單純輸出更大的成熟者。 | **自然壽命盡頭已難以觀測。** |

#### 各階段成立邊界

| 階段 | 已成立能力範圍 | 不成立事項與限制 |
|---|---|---|
| 初顯者／成形者 | 學生訓練、競技、基礎招式建立、失誤、限制理解與打法改良。 | 不得因一次爆發或一次比賽直接跳成成熟／超脫。 |
| 契合者 | 高度穩定運用、靈活改策、專業作戰、教師／成人級處置與帶隊支援。 | 契合者仍未掌握自身能力本質規則；未經指定不得解讀為長壽者。 |
| 成熟者 | 自身能力本質規則之精準運用、長壽者的責任、代價、制度位置與生命本質後續影響。 | 不得解讀為無傷、不死、無代價，或能任意使用他人能力／世界法則。 |
| 超脫者 | 同源世界法則的局部運用、近乎無盡壽命所帶來之世界重量與重大局勢影響。 | 不得常態化、不得任意改寫世界、不得等同無敵或永生不死。 |

#### 成熟者｜能力本質規則與必然長壽

成熟者掌握的是**自身能力得以成立的本質規則**。她／他能理解並穩定控制：自身能力可作用的對象、成立條件、有效範圍、維持方式、反作用與必須正式成立的代價。

成熟不是「招式突然變大」，也不是外部武器替本人提供免費強化；成熟必須建立於本人、異能武裝與生命本質的共同進化。完成此階段者，生命本質已跨越一般尺度，因此**必然成為顯著長壽者**。

- 成熟者只掌握自身能力的本質規則，不因此任意取得其他能力。
- 成熟者可以是攻擊、防護、支援、醫療、偵查、指揮或特殊功能型；長壽不等於每個人都擅長正面破壞。
- 成熟者仍會受傷、衰弱、遭到反制與死亡；生命本質受損時，後果可比肉體傷更嚴重。

#### 超脫者｜同源世界法則與難以觀測之壽命盡頭

超脫者是成熟者之上極少數成立之存在。超脫者不只掌握自身能力規則，更已掌握**與自身能力本質相對應的世界法則**。

- 超脫者只能觸及與本人能力同源之世界法則；不得無因跨至無關法則。
- 「掌握世界法則」不等於完整支配世界所有同類現象；其作用仍須受本人承載、武裝狀態、條件、範圍、持續時間、對抗與代價限制。
- 超脫者的自然壽命盡頭難以觀測，但仍可能因本體破壞、生命失衡、法則反噬、同層力量或主動犧牲而死亡。
- 超脫者必須極度稀少；任何一名超脫者真正進入主線，均應足以影響學院、官方、成熟者群體、重大危機或世界長線局勢。

> 例示只供理解層級：火焰系超脫者可能觸及與燃燒或熱同源之法則；重力系可能觸及與重量、牽引或墜落同源之法則；結界系可能觸及與界線、隔離或內外區分同源之法則。例示不得自動新增至任何角色能力設定。

#### 雙向成長與不可偷渡原則

本人之精神成長、意志選擇、生命承載與實際經歷會影響異能武裝；異能武裝亦會回應這份成長，反饋至本人，使生命本質與能力運用逐步變化。武裝型態、術式、穩定度、同步深化與階段提升，均是雙向成長之外顯，而非外部裝備的單方面升級。

- 不得將「武裝升級」寫成與本人精神／生命本質無關的外掛強化或免費延壽來源。
- 不得將奪取武裝、複製武裝外形或破壞投影，寫成取得／奪走本人之階段、能力本質或壽命成果。
- 臨場突破可以建立新理解、招式轉機或短期負荷，但從初顯／成形／契合跨入成熟，或從成熟跨入超脫，必須有正式設定或正文合法正式成立，不得憑情緒爆發自動成立。

#### 2.2.2 靈力伴生體系、異能武裝分類與進化原則

> 本節承接既有生物／神話來源開放方向，以可獨立組合之分類軸整理承載物、具現方式、來源標記與能力作用範圍；只補正式分類與成立邊界，不替既有角色新增武裝、伴生存在、能力或正史事件。

##### 伴生承載類別

| 類別 | 正式定位 | 最低邊界 |
|---|---|---|
| 異能武裝 | 本人能力之主要伴生具現媒介與戰鬥／術式承載物 | 服從本人靈力、正式階段及投影／本體規則。 |
| 伴生存在 | 與角色靈力、生命本質或武裝共鳴相連之伴生靈、伴生靈獸、靈力穩定伴體或特殊共生存在 | 可具陪伴、提示、支援、協同或依法成立之主戰職能；不自動等同異能武裝、不自動套用持有者五階。霜鈴屬此類。 |

##### 異能武裝主分類

| 主分類 | 具現／操作方向 | 例示方向 |
|---|---|---|
| 器物型 | 以可操持或可指揮器物作為主要媒介 | 刀、杖、法典、鏡、弓、航儀 |
| 裝著型 | 以本人身體為主要承載，透過局部附著、全身覆裝、身體延伸或整體形態變化改變動作、防護、感官、機動與戰法 | 鱗甲、羽翼、爪、角冠、外骨骼、全身戰裝、完整獸化／神話顯身態 |
| 生物型 | 以具生命形態之武裝本體承載主要能力 | 狼、鹿、龍、麒麟、鳳凰等武裝顯現 |
| 群體／群落型 | 以多單位或網絡共同形成能力運作 | 蝶群、蜂群、菌絲、藤網、浮游刃群 |
| 場域／構式型 | 以範圍、陣式、界線、路徑或維持構式為主要媒介 | 護域、星圖、門陣、棋盤、靈紋盤 |
| 複合型 | 兩種以上主分類皆為核心、刪一即破壞打法 | 麒麟法典、龍翼裝甲、蝶群鏡陣、珊瑚護域 |

##### 來源標記與能力本質

- 現實動植物、生態現象、器物象徵與世界各地神話生物均可作為**來源標記**；龍、鳳凰、麒麟、九尾狐、獅鷲、天狗、海妖等名稱、代表外觀與核心性質可直接使用，且可轉化為異能武裝或伴生存在。
- 時間、重力、防護、生命修復、生長、蛻變、共生、幻惑、觀測、導航、鎮護等屬**能力本質／職能軸**；「生命現象」不再作武裝主分類，避免把能力方向誤當外形類別。
- 主分類、來源標記與能力本質可組合但不可互相替代；神話原型不自動授予全部傳說權能、越階成果或法則級效果。
- 一般合法使用不因外形具有生命感或神話來源而固定附帶高危後續影響；限制仍由正式階段、靈力運行、作用條件、相性、場域與反制成立。

##### 裝著型判定與異能武裝進化

- **裝著型判定：** 裝著型不限於局部甲片、羽翼或爪角；只要主要戰鬥者仍為本人，武裝透過本人身體的覆裝、延伸或整體變化發揮能力，即可成立為裝著型。可離身獨立行動並作為主要能力媒介之生命形態武裝，才歸生物型；若本人裝著與離身生物媒介兩者皆不可刪除，才歸複合型。
- **全身變化邊界：** 全身戰裝、獸化戰態、神話顯身態或元素／特殊軀體態可作初始武裝形態，也可作正式進化結果；其成立不自動代表升階、永久非人化、失控人格或取得來源原型之全部權能。
- **進化原則：** 異能武裝之進化遵循「渴望定向・意志塑形・職能可轉化」。深層所向決定長期進化方向；角色經由正文實際正式成立之持續選擇、代價及重大場面中的真實意志，可觸發共鳴徵兆、用途展開、職能轉化、形態進化或能力本質深化／轉向。
- **職能轉化邊界：** 進化所延續者首先是角色真正想完成或守住之事，而非現階段表面職能；因此防護可為主動排除威脅、守住同伴而發展攻擊／壓制型態，攻擊亦可為護送或守住他人而展開防護／守域用途。新方向必須可由既有武裝、能力來源、人物所求及已正式成立選擇形成因果，不得因一時情緒、臨場翻盤或作者便利無因追加效果。
- **保存與升階：** 武裝外形改變、全身變化、用途展開或職能轉化不自動等於正式升階；候選正文中首次出現之進化或型態，未經使用者明確正式採用並完成結算，不得寫入正式能力資料或作後續承接。

#### 2.2.3 高科技靈力醫療、治療型武裝與生命復歸

本世界之醫療以成熟高科技與靈力科技共同運行；醫療設備、生命監測、靈力調整與治療型異能武裝互相支援，但不互相取代角色價值。下列規則建立世界硬邊界、能力上限與短期傷勢之活性清潔尺度：不回溯增寫既成治療場面，但普通小傷超出合理處理窗口且無特殊延長原因時，不再保留為當前活性限制。

| 層級／問題 | 正式裁定 | 必守邊界 |
|---|---|---|
| 普通傷勢與一般負荷 | 日常創傷、一般靈力耗損、投影測試負荷、正式競技中可控之外傷與精神震盪，原則上可由校內／都市醫療系統、恢復設備或合法治療能力依法修復或調整。 | 傷勢於當場及合理短期內可限制出力或選擇；不得無理由跨日／跨章保留。需延長者，正文必須成立特殊不穩、治療受阻、高精度穩定問題、本體／靈魂層傷害或新傷來源。 |
| 後續影響保存 | 身體修復不等於事件消失；已錯失場次／資格、已暴露打法、已形成公開紀錄、關係或制度後續影響仍為正式事實。 | 不以快速醫療抹除勝敗、選擇或已成立後果。 |
| 治療型異能武裝 | 具治療、生命修復、再生或回復職能者，能實質修復傷口、組織、骨骼、生命狀態、靈力紊亂及依法成立之特殊損傷。 | 不得無因弱化為只止血、只穩定或必須等待設備接手；上限依正式階段與能力卡。 |
| 自我治療 | 治療型能力原則上可作用本人與他人；除正式機制另有明定，自療不預設失效或較弱。 | 自療仍受治療窗口、靈力、集中、多目標選擇、持續受擊與特殊干擾限制。 |
| 本體／靈魂本源創傷 | 具修復可能，但難度、資源、所需位階、恢復時間與生命風險遠高於普通傷勢。 | 不寫成一般醫療即可無痕修復，也不預設必然永久失能。 |
| 生命復歸 | 肉體死亡已成立，但靈魂本源尚未消散且未遭徹底毀滅時，可由極高階治療型異能武裝、高科技靈力醫療設備與必要協作，在有限窗口內依法救回。 | 需本源狀態、肉體可重建性、介入窗口、治療位階、設備／支援、干擾解除與救回後後續影響共同成立；不得作無條件復活或臨場便利翻盤。 |
| 常規不可救回邊界 | 靈魂本源完全消散或遭徹底毀滅者，原則上不屬一般治療或生命復歸可救回範圍。 | 若未來欲突破此邊界，須另行正式建立超脫者級或世界法則級例外。 |

##### 普通小傷之活性清潔尺度

- 普通麻感、輕微肩腕負荷、一般投影測試負荷與可控精神震盪，只在當場或合理短期內作為精度、上場與選擇限制；超出該時間尺度且無特殊延長原因者，自當期運行盤與活性繼承中清除。
- 清除活性限制不等於回溯改寫正文或增寫治療場面；角色曾受傷、曾因狀態錯失場次或曾以低負荷完成行動之事件事實仍保留。
- 沙耶右腕短線已由第八、九章依法正式成立並結案；現行不得列為待確認或活性 `P0`，若未來有新限制，須由新正文另行成立。

##### 治療型能力與正式階段

| 階段 | 合理治療範圍 | 防錯 |
|---|---|---|
| 初顯者 | 小傷癒合、止血、緩痛與輕度靈力穩定。 | 不處理死亡復歸或重大重建。 |
| 成形者 | 快速修復一般傷勢並實質處理中度創傷、骨折、失血與靈力紊亂；可自療；在條件充分時參與重傷搶救。 | 不單獨完成肉體死亡後復歸；不能無代價處理極高污染／靈魂重創。 |
| 契合者 | 依戰況重組治療方式，處理重傷、臟器損傷、污染侵蝕、多人戰地救援與瀕死搶救；極佳條件下可參與復歸流程。 | 復歸仍非輕易單獨常態效果。 |
| 成熟者 | 掌握生命／治療本質規則者，可承擔重大組織重建、本體／靈魂共鳴修復，並成為生命復歸之主要執行者。 | 仍受本源狀態、窗口、設備、干擾與後續影響限制。 |
| 超脫者 | 可在同源生命法則與正式條件內處理更高層級之存續、本源重建或復歸問題。 | 不自動救回本源已完全消散／毀滅者；不得無限制改寫生死。 |

### 2.3 異能武裝靈魂內收納、召喚與維持準則

#### 2.3.1 能力本源／伴生武裝／失能層級裁定

| 層級 | 定義 | 戰鬥／比賽可成立之影響 | 不成立事項與限制 |
|---|---|---|---|
| 本源能力層 | 使用者靈魂與生命本質所具有之固有能力來源。 | 即使武裝失位，仍是角色本人具備之根本；具體可見表現依個別設定。 | 不因單純武裝離手直接歸零；不由他人拿走武裝即轉移完整本源。 |
| 武裝具現層 | 本源能力凝成之伴生武器／媒介型態。 | 支撐招牌戰法、精準控制、較高輸出、較大範圍、長維持與武裝專屬術式。 | 不得被降成純裝飾；不得無因讓未具現狀態等同完整具現戰力。 |
| 媒介受損／失位層 | 武裝被打離、握持破壞、具現受阻、遠端連結被干擾或被迫收回。 | 可迫使角色換打法、降低精度／火力／維持、失去特定招式啟動條件或承擔逆轉風險。 | 不等同能力核心失效，除非另有更深層干涉成立。 |
| 本源抑制／失能層 | 作用於本人靈力、靈魂共鳴、精神、生命本質、能力核心或明確特殊封禁。 | 可造成真正失能、斷聯、失控或高額代價，屬重大戰鬥／危機條件。 | 不得低成本、無痕、無代價頻繁使用；不得由普通奪武替代。 |

#### 2.3.2 競技中之媒介受力與能力邊界

1. 競技中可依法成立奪武、逼退、封住媒介與中斷具現等戰術；其實際效果依個別能力、場地與安全規則判定，不自動結束對局。
2. 武裝離手後可否使用基礎能力、維持已展開效果，或以何種代價反擊，均依個別正式能力設定與當下狀態成立，不由共通規則預填。
3. 比賽安全規則若限制武裝型態、輸出或致命術式，只限制依法成立之競技使用邊界，不改寫角色能力本源。

#### 2.3.5 正式競技之決勝、傷害與裁定底線

1. **正式競技無安全得分勝負：** 校內正式選拔與全國代表競技，不採「先碰到指定部位即得分」「累積有效接觸」「奪標入圈得分」「三回先取二」等測試格式。比賽必須進行至一方認輸、失去續戰能力、被醫療／裁定強制中止、失格，或多人賽中一方已無合法續戰成員。
2. **時間規格：** 正式單局原則無固定短時限；若因場地安全、賽程事故或重大中止條件而停止，須依法裁定，不以時間到後比小分取代真正勝負。
3. **個人戰：** 一方認輸、失去意識、無法保持戰鬥行動、武裝／本源狀態不足以續戰、遭醫療強制中止或失格時，另一方獲勝。
4. **雙人／團體戰：** 一方全體有效參賽者認輸、失去續戰資格、遭裁定中止或失格時，對方獲勝；個別成員失能會改變場上人數與配置，不自動終止整場。
5. **真實傷害：** 本體顯現之正式競技接受肉體傷害、骨折、失血、器官損傷、靈力失衡、武裝損害、本源創傷、退賽風險乃至死亡可能進入因果。不得因角色重要、朋友關係或校園背景而自動降為手麻、擦傷或檢查無事。
6. **醫療與裁定功能：** 場邊醫療、結界與裁定員負責辨識續戰可能、強制中止過度傷害、搶救與保存後續責任；不負責將比賽改寫為無傷安全遊戲，也不撤銷已成立之敗北、傷勢、失位或公開後果。
7. **競技底線：** 正式競技之合法目的為擊敗對手、迫使認輸或使其失去續戰能力；勝負已分、裁定中止或對手已無抵抗能力後之追擊、處決性攻擊與蓄意擴大不可逆損害，屬違規／犯罪並依法追究。此底線不保證交戰過程不存在致命後果。
8. **兒童與低風險邊界：** 幼稚園／小學學生、安全課、普通日常競技與另行明示之低風險活動仍限投影顯現，不因正式學生賽制改為本體而擴大其高危參戰權限。
9. **危機越界：** 真實敵襲、不明組織介入、重大危機或高危救援若介入賽場，該場面不再只以競技規則判定；危機造成之傷害、死亡、搶救、責任與後續局勢均須另依法成立。

#### 2.3.6 具生命／神話來源、全身裝著與武裝進化運作邊界

1. 具生命或神話來源之**異能武裝**，其具現、維持、投影／本體、失位與受損方式仍服從本人靈力及異能武裝規則；可依法移動、護衛、追擊、纏繞、飛行、引路、分群或展域，但只限已成立能力軸。
2. **裝著型異能武裝**可由局部附著、全身覆裝、身體延伸或本人整體變化成立；完整獸化、神話顯身或元素／特殊軀體態，只要戰鬥主體仍為本人，仍屬裝著型。其體型、重心、感官、動線、場地適性與本體受創風險須依法可讀；不因外觀完整便自動升階、失控或取得原型全部權能。
3. **伴生存在**為平行承載類別；其常態是否在場、如何與本人或武裝共鳴、是否可主動行動或參戰，皆依個別資料，不自動套用異能武裝之召喚／收回／離手／投影破壞規則。
4. 若戰鬥涉及本人裝著、生物型武裝與伴生存在同場協作，須區分主要能力媒介、可分位行動者、提示／支援／協同來源及各自受力後果；不得把伴生存在當作未建檔新武裝，也不得將裝著型本人變化誤寫成無來源獨立生命。
5. 神話原型的特殊性須落在本作可操作能力與正式職能上：例如龍型究竟偏吐息、鱗甲、引流或壓場；麒麟來源究竟偏鎮護、淨化或穩場；不得只憑名稱臨場補得未建立效果。
6. 異能武裝進化可依法使型態、用途、職能或能力本質深化／轉向；職能可跨越原表面定位，但須延續角色深層所向並由正式正文中的選擇與代價正式成立。進化不得被用作無來源翻盤、無代價強化、自動升階或候選內容滲漏成史。
7. 具體角色若正式成立伴生存在自主性、共生負荷、群落生命尺度、裝著型全身形態、武裝進化結果、外觀生命特徵或高階生命法則，依法同步其個別資料；在此之前，本節只提供分類、進化與成立邊界。

#### 5.16.2 能力本質

《未竟折門》的能力不是時間倒流，不是因果歸零，不是絕對防禦，不是復活，也不是現實改寫。

它的本質是「結果阻斷」。

被未竟化干涉的行動仍然會發生，角色仍會移動，靈力仍會消耗，部分場地痕跡、壓力、副作用與前置傷害仍可能保留。

但最終結果不被承認。

例如：

- 攻擊已經揮出，但命中不成立。
- 訊息已經輸入，但送達不成立。
- 治療已經執行，但治療完成不成立。
- 封存盒已經交到手中，但交接完成不成立。
- 撤離已經抵達出口前，但撤離成功不成立。
- 報告已經送出，但系統仍顯示未提交。

---

#### 5.16.3 啟動條件與限制

| 規則 | 內容 |
|---|---|
| 必須存在流程 | 目標行動必須能被拆成「開始／進行／接近完成／完成結果」。 |
| 必須存在完成點 | 行動必須有明確成功、失敗或完成判定，黎久瀨才容易干涉。 |
| 必須辨識完成句 | 黎久瀨需要抓到對方想完成什麼。完成句可以是語言、動作意圖、流程痕跡、通訊格式或系統判定。 |
| 對明確目標強 | 對「打中黎久瀨」「送出訊息」「完成治療」「完成封存」「抵達出口」等明確目標極強。 |
| 對模糊目標弱 | 若對方沒有固定完成點、行動即興、目的拆散、或不追求系統承認，未竟化效果會下降。 |
| 不是時間倒流 | 被阻斷後，消耗、位移、壓力、前置痕跡與部分副作用仍可保留。 |
| 不能無限重複阻斷 | 同一目標、同一結果若反覆阻斷，黎久瀨自身負荷與反噬會上升。 |
| 大範圍結果較難折回 | 多人見證、外部系統承認、紙本物證固定、身體已承受完成結果時，未竟化會變弱。 |
| 代價會回流 | 黎久瀨越常讓別人無法完成，自己的行動也越容易延遲、卡住或停在未完成。 |

#### 5.16.4 主要招式

| 招式 | 類型 | 效果 |
|---|---|---|
| 折步 | 流程錯位 | 將流程中的某一步拆到錯誤位置，使後續順序錯亂。 |
| 未竟門檻 | 完成阻斷 | 讓目標行動在完成前一刻停住，結果無法成立。 |
| 缺頁程序 | 條件抽離 | 抽走流程中的關鍵條件，使對手不知道缺了哪一步。 |
| 延遲終點 | 終點拖延 | 讓出口、距離、時間或完成條件向後延長。 |
| 錯位回軌 | 結果錯投 | 讓完成結果送到錯誤位置，使系統、本人或見證者收不到完成證明。 |
| 半開折門 | 門域展開 | 展開門域，將場內所有行動拆解為未完成流程。 |
| 未竟拒收 | 有限自動防禦 | 在特定條件下，將直接導致黎久瀨死亡、重傷或拘束完成的結果拖入未竟狀態。不可無限使用。 |

#### 5.16.5 半開折門場

| 欄位 | 內容 |
|---|---|
| 展開名稱 | 半開折門場 |
| 場域效果 | 將場內行動拆成「開始／進行／接近完成／尚未完成」四段。 |
| 核心干涉 | 隱藏真正的完成點，使行動永遠停在完成前一刻。 |
| 對戰效果 | 對任務提交、通訊、治療、撤離、封存、交接、命中判定極強。 |
| 維持代價 | 消耗高，維持時間有限；展開越久，黎久瀨自己的行動也越容易卡在未完成。 |

#### 5.16.6 反制方式

| 反制方式 | 說明 |
|---|---|
| 建立旁路完成 | 不走被黎久瀨掌握的正式流程，改用現場能成立的替代完成條件。 |
| 多人同步見證 | 讓結果不只依賴單一系統、單一確認欄或單一見證者。 |
| 封回缺頁條件 | 找出被抽走的關鍵步驟，補回流程缺口。 |

| 破壞媒介 | 攻擊半開門框、流程板、確認欄、資料窗等媒介，而非直接攻擊黎久瀨本人。 |
