# Character Voice Registry v0.1

Phase23B：核心主角群性格／聲線 Registry

## 安全聲明

本檔為 Character Voice Registry read-only derived index，用於整理角色性格、聲線與寫作檢索輔助。

- 本檔不高於 Canon DB。
- 本檔不得覆蓋 active_engine。
- 本檔不得新增正史。
- 若本檔與 Canon DB / active_engine 衝突，以 Canon DB / active_engine 為準。
- 使用者 Phase23B 補充資料可作為 Registry 草案來源。
- 正式寫入 Canon 仍需走採用、結算與 approval gate。
- 未明確欄位必須標記「未明確／不可補猜」。

## 禁止誤寫總則

- 不得把「冷靜」角色全部寫成同一種冷面。
- 不得把「外向」角色全部寫成吵。
- 不得把「內向」角色寫成沒有主見。
- 不得把兒童角色寫成成人級主戰力。
- 不得把支援角色寫成工具人。
- 不得把異能武裝定位直接當作完整性格。
- 不得把使用者補充資料直接覆蓋 active_engine；本檔僅作 Registry 草案索引。
- 缺資料欄位必須標記未明確，不可補猜。
