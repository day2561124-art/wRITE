# 二、正文總成稿方式

## 1. 先寫人物眼前真的要處理的事

每章先成立一件人物此刻無法忽略的具體事情：

- 他想爭取、保住、避開或確認什麼；
- 他本來要去做什麼，卻被什麼打斷；
- 另一人為什麼會介入，且介入後自己也要承擔什麼；
- 事件結束後，誰的下一步因此變麻煩。

正文不先宣布人物變得有魅力、被注意、靠近彼此或找到位置；這些只能由行動、勝負、代價與後續被讀者看見。

## 2. 每章支付一件真正有重量的事

一章不必同時推進所有人物線、競技線、關係線與危機線。起稿前只需確定：

- 本章主要事件是什麼；
- 這件事改變了誰的當下位置；
- 它比近期已成立事件多了什麼新壓力；
- 章末仍留下哪一個具體後果。

其他長線若自然經過，只留必要痕跡，不搶走主戲。

## 3. 骨架目標必須轉成事件

| 骨架目標 | 正文成立方式 |
| --- | --- |
| 建立魅力 | 人物在限制、衝突或勝負中做出具有本人性的選擇，並承擔代價。 |
| 找到位置 | 人物主動爭取、拒絕、承受失敗或放棄某個實際位置。 |
| 慢熱靠近 | 一次實際偏向、保留、拒絕、失去便利或替對方承擔不方便。 |
| 被注意 | 具體表現造成外界實際反應、針對、排程或判斷改變。 |
| 危機浮出 | 出現可感、可追查、會影響生活或競技的具體人為問題。 |
| 留下後果 | 身體、時間、名額、下一場安排、相處方式或判斷確實改變。 |

# 三、章節承重、群像調度與剪輯支付層

本層用於防止章節只完成流程、展示規則、鋪氣氛或延後支付。每章不必解決所有長線，但必須讓至少一件人物位置、關係、身體、排程、資訊或世界摩擦發生可承接的具體變化。

## 1. 每章必須有承諾、推進與支付

起稿前先確認本章對讀者的最低承諾：一場具名對局、一個關係選擇、一項後果、一個危機痕跡、一段生活麻煩或一個人物位置改變。

正文中段須讓該承諾持續被推進；章末須支付至少一項具體結果。支付不一定是大勝負或大真相，但必須是角色下一步無法當作沒發生的變化。

不得只用公告、旁觀、討論、氣氛、預告或設定展示代替支付。若本章刪除後，後續只少了氣氛、不少實際位置變化，則本章主戲不成立。

## 2. 群像章要讓生活軌道互相干擾

群像不是讓多人集合關心同一件事，而是讓兩條以上人物生活軌道互相撞歪。

可用的干擾來源包括：

- 場次、檢錄、觀戰區、醫療門禁；
- 社團登記、訓練排程、器材歸還；
- 通訊延遲、終端限制、靈軌時間；
- 家庭安排、外校支援、教師或裁定通知；
- 傷勢複查、候補替換、公開紀錄與輿論。

承重出場者應帶著自己的時間壓力、目的、面子或不願失去的東西進場，不得只因主角事件而集合。

## 3. 對手與反派須同時造成外部壓力與內部對照

承重對手不只阻止主角達成目標，也應逼出主角的習慣、弱點、偏心、恐懼或錯誤判斷。

正式競技對手即使只出場一章，也須具備：

- 自己想贏或不能輸的理由；
- 可看見的武裝、打法、強點與限制；
- 能傷到主角或逼主角改策的位置；
- 對主角內部問題形成壓力的功能；
- 勝後或敗後仍屬於本人的後續。

反派或危機方不得只以制度、表單、抽象不安或遠端黑幕存在；其壓力須落為具體人物、物件、行動、利益、設備、路線、受害者、失物、封口、威脅或可追查痕跡。

## 4. 世界摩擦要製造選擇成本

世界細節必須以生活摩擦形式進場，而不是單純展示設定。

制度、靈力科技與城市設備可造成：

- 門禁讓人進不去；
- 醫療規則限制陪同；
- 觀戰區製造資訊差；
- 社團登記卡住時間；
- 靈軌與通訊造成錯過；
- 公開紀錄改變他人判斷；
- 裁定與醫療摘要影響下一場安排。

制度與科技不得取代人物選擇；它們應製造選擇成本、資訊不對稱、後座或人際摩擦。

## 5. 章節可剪輯，但每個場景都要有切點

正文不必從事件起點寫到所有善後結束，也不必接續上一章最後一秒。可跳接、切場、略過低承重過程。

每個場景都應有明確進場理由與離場切點。當人物完成一個新選擇、資訊差形成、身體後座落地、關係偏移、場次結果支付或下一個麻煩出現，即可切場。

不得用流水帳保存所有移動、等待、吃飯、回教室、群組反應與無效善後。若場景沒有讓人物位置、資訊、關係、身體或排程改變，應刪除或併入其他場景。

## 6. 完稿前五問

正文完成前，須在內部通過五問：

1. 本章若刪掉，後續人物位置、排程、關係或資訊是否會少一個必要改變？
2. 本章主戲是否由具名人物自己的目的與選擇推動，而不是公告、制度或旁白推動？
3. 本章是否至少有一項勝負、傷勢、失敗、錯過、偏心、資訊差或生活麻煩被支付？
4. 本章是否讓至少一名角色暴露了只有他本人會有的判斷、弱點或選擇？
5. 本章章末是否仍有具體後座，而不是被漂亮句子收乾淨？

任一答案為否，應在內部重選主戲、重寫場景承重或改換章末切點。

# 四、潛台詞、視角焦點與讀者感受層

本層用於防止正文變成全知報告、直白情緒說明、流程紀錄或平均敘述。每場承重戲都必須同時存在「表面事件」與「內在拉扯」：表面事件是比賽、登記、觀戰、醫療、通訊、社團或日常麻煩；內在拉扯是人物不願說破的害怕、偏心、面子、誤會、嫉妒、失敗感、責任或逃避。兩層必須互相推動。

## 1. 每場都要有微張力

微張力不是大危機，而是場面裡持續存在的小不順、小矛盾或小代價。

可用來源包括：

- 話語與真意不一致；
- 行動與宣稱不一致；
- 想幫忙但有成本；
- 想逃避卻被留下；
- 想取勝卻怕暴露弱點；
- 想關心卻只能問技術、時間、物件或規則；
- 角色明明知道答案，卻選擇不說完。

沒有微張力的日常容易變成資訊整理；沒有微張力的戰鬥容易變成招式流程；沒有微張力的戀愛容易變成甜場展示。

## 2. 承重場景須有行動、反應與決定鏈

每個承重場景不能只讓事件發生，還要讓人物因事件產生反應，並改變下一步。

場景最低因果鏈為：

> 行動／事件發生 → 人物反應 → 新決定或新麻煩

反應可以是留下、離開、改場次、改說法、隱瞞、追問、放棄、誤判、承擔後座、改變站位或拒絕被照顧。若場景只完成資訊展示，沒有改變人物下一步，應刪減、合併或重寫切點。

## 3. 情緒優先以潛台詞呈現

關係、嫉妒、擔心、不甘、恐懼、歉意與偏心，優先藏在人物沒有說出口的地方。

可用方式包括：

- 角色用技術、時間、場次、登記、傷勢處理或無關小事代替真正情緒；
- 訊息打完又刪掉；
- 避開稱呼或突然改稱呼；
- 重複確認已經知道的資訊；
- 明明能走卻多停一拍；
- 多做一步行動，嘴上卻否認在意；
- 把道歉、關心、害怕或嫉妒換成不漂亮的句子。

不得讓角色頻繁直接說出完整自我理解、完整關係進展或本章主題答案。

## 4. 視角焦點不可寫成全知報告

每場承重戲須明確選擇主要感知焦點。正文優先呈現視角人物能看見、聽見、誤判、忽略、在意或不願承認的內容。

同一場正式競技中，不同角色的焦點應不同：

- 戰鬥者看見距離、節拍、武裝軌跡與受力；
- 支援者看見傷勢風險、保護窗口與誰被暫時放棄；
- 旁觀者看見公開觀感、資訊差與自己站不到場上的位置；
- 對手看見可利用的習慣、破綻與反制條件。

若切換視角，必須帶來新的資訊差、情緒差或判斷差；不得只用另一個人重述同一場面。

## 5. 承重角色須有情緒弧

承重角色在本章須有可追蹤的情緒位置變化：

- 開場狀態：他現在想避開、保住、證明或否認什麼；
- 中段受壓：事件如何逼出他的弱點、偏心、錯判或不成熟反應；
- 章末殘留：他變得更煩、更沉默、更不甘、更偏心、更不會面對，或做出一個不能當作沒發生的選擇。

情緒弧不等於成熟、和解或頓悟；角色可以因事件變得更不安、更嘴硬、更逃避。不得讓事件結束時所有情緒被旁白整理乾淨。

## 6. 世界設定要回到人物習慣

制度、城市、學院、社團、交通、通訊與醫療，不只作背景展示，也要影響角色怎麼安排時間、相信誰、避開誰、求助誰、違規到哪一步。

世界細節應改變人物行動或判斷，例如：

- 醫療門禁讓某人不能陪同，只能留在外面；
- 觀戰區規則造成資訊差；
- 社團登記讓想留下的人不能留下；
- 靈軌時間讓同行變成選擇，而不是便利；
- 通訊限制讓未回覆本身成為情緒訊號。

若世界細節不改變人物行動或判斷，應刪減或轉為背景淡寫。

## 7. 句子節奏須服從場面功能

戰鬥、沉默、群像、公告、醫療、戀愛與日常不應使用同一種敘述密度。

- 戰鬥突發可用短句、近鏡頭與身體反應；
- 戀愛猶豫可用留白、停頓、沒說完與錯開話題；
- 群像公共場景可用多聲源、動作插入與不完整資訊；
- 公告、裁定與系統語句須清楚可執行，不要詩化；
- 傷後日常應在普通動作中保留不方便。

若所有段落都呈現同樣長度、同樣抒情度、同樣旁白解釋量，須重寫節奏。

# 五、伏筆資訊、場景波峰與長篇修稿層

本層用於避免長篇正文只累積氣氛、設定與事件清單，而沒有形成讀者可追、角色可承擔、後續可支付的連載推進。伏筆、資訊差、場景波峰與章末未解狀態，皆須服務人物行動與後續承接，不得只作神秘感或漂亮收尾。

## 1. 伏筆必須有支付距離

伏筆不是抽象不安，也不是無指向異常。正文可留下低濃度暗線，但每個伏筆至少須能對應一項未來支付方向：人物、能力、組織行動、制度縫隙、場地問題、關係後果、資訊差或物件異常。

伏筆分三種：

- 近程伏筆：一至三章內應有反應、小支付或反向確認；
- 中程伏筆：篇章內應逐步累積，不得長期只重複同一異常感；
- 長程伏筆：可只露痕跡，但必須保持為未支付方向，不得被正文寫成角色已知答案或已成立真相。

不得只寫「不對勁」「像有什麼在看著」「紀錄怪怪的」卻沒有可追查對象、位置、物件、制度或人物行動。

## 2. 承重場景須推到場景波峰

每個承重場景須有一個場景波峰：人物原本的姿態、計畫、隱瞞、距離、節奏或判斷被推到無法完全維持的位置。

波峰不一定是吵架、受傷或爆炸，也可以是：

- 某人終於沒接住一句話；
- 某人改掉原本的登記時間；
- 某人被迫承認自己看見了不該忽略的細節；
- 某個能力徵象沒有照常消散；
- 某個場地規則讓人不能照原計畫靠近。

波峰後須揭露至少一項新資訊：人物選擇、能力限制、關係偏移、世界規則、對手策略、伏筆痕跡或下一步麻煩。不得讓場景只以平穩對話、資訊交換或氣氛停頓結束。

## 3. 每章須保留一條「想要／現實阻力」張力線

人物想要的東西可以很小，但必須具體：趕上場次、避開追問、保住面子、看完一場比賽、不要被照顧、拿回物件、把訊息送出去、讓對方不要發現自己在意。

現實阻力也必須具體：名單、門禁、醫療規則、場地距離、社團登記、觀戰區、人群、傷勢、對手情報、通訊限制、靈軌時間、家庭或學院制度。

不得只以「心情複雜」「不知道怎麼辦」「氣氛尷尬」當作主要阻力；抽象情緒須落回可見行動、物件、時間或制度摩擦。

## 4. 資訊須分成讀者知道、角色知道與角色以為自己知道

每章須注意三層資訊：

1. 讀者知道什麼；
2. 場上角色知道什麼；
3. 角色誤以為自己知道什麼。

資訊差應推動行動：誤判、追問、隱瞞、錯過、改排程、站錯位置、信錯人、沒有及時提醒或選擇不說。不得讓公告、旁白、教師解說、終端訊息或旁觀者評論一次消除所有資訊差。

若切換視角，必須增加新的資訊差、情緒差或判斷差；不得只換視角重播相同場面。

## 5. 重要場地須累積情感與事件痕跡

重複出現的場地不得只是背景板。訓練館、公共廳、公告牆、醫療門外、影術棋盤社開放桌、靈軌站、教室後排、社團倉庫、觀戰區等地點，應逐步累積人物記憶、勝敗、錯過、等待、關係偏移、制度壓力或公開觀感。

再次回到同一場地時，人物行動與感受應受前次事件影響；不得每次都像第一次使用該場地。場地細節若不改變人物行動、情緒、資訊差或後座，應淡寫或刪減。

## 6. 章末須留下具體未解狀態，不以抽象懸念收尾

章末可以留下未回答問題，但必須是具體未解狀態，不是抽象懸念或漂亮總結。

適合章末保留的狀態包括：

- 未回覆的訊息；
- 改掉的場次或登記；
- 醫療門外仍等著的人；
- 沒有收回的武裝徵象；
- 對手留下的觀察或針對；
- 觀戰區或公告牆造成的公開觀感；
- 能力、物件、制度或關係上的未解麻煩。

不得只以「她不知道接下來會怎樣」「事情還沒結束」「風還在吹」之類抽象句作為懸念。

## 7. 候選正文完成前須按分層順序修稿

內部修稿須先修故事骨架，再修句子，不得保留壞因果只把文字磨漂亮。

修稿順序如下：

1. 結構層：本章承諾、推進、支付是否成立；刪掉本章後，後續是否會少一個必要變化。
2. 場景層：每場是否有目標、想要／阻力、場景波峰、切點與具體後座。
3. 人物層：角色是否因本人目的、弱點、偏心、面子與選擇行動。
4. 資訊層：讀者知道、角色知道、角色以為自己知道三層是否清楚，資訊差是否推動行動。
5. 能力／戰鬥層：能力媒介、條件、反制、代價與勝負因果是否可讀。
6. 語言層：刪除旁白總結、流程語、硬解釋、過度漂亮句、同質化節奏與抽象懸念。

任一核心層不成立，須重選事件、重寫主戲或改切點；不得只做句子清潔。

# 六、長篇連載可追讀性層

本層用於防止長篇群像在章數累積後變得散、滿、膩、難追或固定角色壟斷。共同主角地位平等不等於每章平均出場；原創拓展自由不等於每章大量新增必記資料；長線伏筆存在不等於支線可以長期空轉。每章須讓讀者知道本章為何重要、需要記住什麼、哪些人暫時不在場但仍活在世界裡。

## 1. 群像輪替不追求章章平均，而追求長線公平

每章可聚焦少數人物，但須避免連續多章只讓固定小隊壟斷承重。角色缺席不等於消失，角色出場也不等於有戲。

處理原則：

- 出場者須有自己的目的、時間壓力、面子、利益或不願失去之物；
- 缺席者若與本章事件相關，可用一句具體動線交代其場次、社團、醫療、家庭、外校或城市安排；
- 不必為了公平硬塞全員反應；
- 不得讓同一批角色長期承擔所有勝負、發現、救援、關係推進與章末收束。

群像公平是長線輪替，不是單章分配篇幅。

## 2. 支線須留下下一個可見台階

任何被本章啟動、推進或重新點亮的支線，都須留下下一個可見台階，而不是只留下「以後很重要」的空泛感。

可見台階包括：

- 下一場對局、下一次檢錄、下一次觀戰或下一場改排程；
- 下一次醫療複查、社團開放桌、器材歸還、通訊回覆或外校接觸；
- 下一個該知道的人、下一個要處理的物件、下一個需要再去的場地；
- 下一個會發酵的誤會、資訊差、關係偏移或對手觀察。

支線可暫時不支付結果，但不能沒有下一步形狀。

## 3. 新增資訊須節制，區分必記與背景質感

候選正文可新增新角色、臨時對手、場地、制度、能力徵象與世界細節，但每章新增可記憶資訊須節制。

本章必記資訊包括：

- 新勝負、新傷勢、新場次變動；
- 新能力徵兆、新關係偏移、新支線台階；
- 新對手的名字、武裝、打法限制；
- 新伏筆方向或會影響後續承接的制度／物件變化。

背景質感資訊包括：

- 場地氣味、人群聲音、路線細節、普通設備、臨時職員、城市生活痕跡；
- 只服務當場氛圍與動線、不要求讀者後續記住的低風險細節。

原則上，每章必記新增資訊不宜超過三項；若超過，須合併、延後或改為背景淡寫。不得把引擎可新增世界細節誤用成設定展示。

## 4. Stakes 須具體化，不只靠死亡或大危機加壓

每章不必都升到死亡、重傷或主線危機，但必須明確角色可能失去什麼。

可失去的東西包括：

- 名額、場次、準備時間、體力、武裝穩定度；
- 面子、主導權、觀眾評價、對手情報差；
- 原本距離、信任、被照顧或不被照顧的權利；
- 下一場安排、社團位置、搭檔默契、自我認知。

若角色失敗後什麼也不會少，勝利後也沒有任何負荷或位置改變，本章壓力不足，須重建 stakes。

## 5. 重大事件宜形成場景序列

正式對局、傷勢、能力徵兆、關係偏移、暗線痕跡與重大失敗，若承重大，不宜單場塞滿後立即收乾淨。可採三段式連鎖：

1. 前置壓力：人物想要、現實阻力、資訊差或生活軌道先被撞歪；
2. 當場支付：勝負、受傷、錯過、失言、能力徵象或制度後座正式落地；
3. 後座落地：醫療、公告、社團、通訊、觀戰、關係或下一場安排承接結果。

三段可在同章完成，也可跨章支付；但每一段都須有當下可讀價值，不得只作過渡。

## 6. 重複場地與同類場面不得重播同一功能

公告牆、觀戰區、訓練館、醫療門外、公共廳、影術棋盤社開放桌、靈軌站、教室後排等場地可重複出現，但每次使用須多一層人物或事件變化。

例：

- 公告牆第一次可公布資訊，第二次應造成誤讀、位置改變、缺席或公開壓力；
- 觀戰區第一次可建立正式競技危險，第二次應製造資訊差、偏心、逃避或關係後座；
- 醫療門外第一次可等待，第二次應暴露權限、不能進去的代價、被排除感或誰被允許靠近；
- 社團桌第一次可建立關係距離，第二次應改變登記、座位、物件或誰願意留下。

同類場景若只能重複上一章功能，應換場、換視角、換事件或刪減。

## 7. 章節開頭優先從變動點切入

章節開頭不必習慣性從完整日期、天氣、集合、公告流程或上章下一秒開始。優先從人物已經遇到的不穩點切入：

- 名字被念到第二次；
- 場次提前、座位被佔、門禁不開；
- 傷勢通知抵達、對手先到、某人缺席；
- 物件位置不對、通訊沒回、公告顏色被改；
- 原本應該在場的人不在，或不該在場的人先到了。

若從平穩流程開始，該流程本身必須已經失常，或能在短時間內撞出人物選擇。

# 七、聲線差異、轉場負荷與清稿去模板層

本層用於防止正文在規則正確後仍讀起來像流程、摘要、設定整理或同一種 AI 聲音。它不新增正史、不改角色設定，只規範成稿時的聲線、轉場、意象與清稿品質。

## 1. 角色聲線不得只靠口癖區分

角色聲線由關注順序、詞語密度、反問方式、沉默方式、處理尷尬的方法與身體動作共同構成，不只靠口癖、句尾或固定短句。

成稿時應確認：

- 遮掉說話者名字後，承重角色仍能由語氣、關注點或動作辨認；
- 熟人對話不反覆講完整專名、背景與彼此已知資訊；
- 角色不會因需要交代設定而突然使用相同的說明腔；
- 角色的沉默也應有差異：有人逃避、有人觀察、有人生氣、有人不知道怎麼接話、有人正在計算下一步。

若所有角色都用同一種句長、同一種反問、同一種幽默與同一種理解方式說話，須重寫對話與動作節奏。

## 2. 對話須混入有效動作，不用台詞講滿情緒

對話場面不得只靠台詞互丟推進。人物正在逃避、擔心、吃醋、不甘、想靠近或想保住面子時，優先讓他處理物件、改變站位、刪改訊息、看錯地方、打斷話題、重複確認小事或多做一步。

可用的有效動作包括：

- 收起又拿出的終端；
- 沒遞出去的糖、飲料、外套或繃帶；
- 被推近半寸的椅子、登記簿、棋片或入場證；
- 站在門外但不進去；
- 沒有糾正對方放錯的東西；
- 明明要走，卻先把對方的器材放好。

不得讓角色頻繁直接說出「我擔心妳」「我其實在意」「我不知道怎麼面對」「我們好像更近了」之類完整情緒答案。

## 3. 轉場負荷須受控，切場必須帶來新價值

章節可跳接、切場與略過低承重流程，但不得頻繁要求讀者重新定位。每次轉場都應帶來至少一項新價值：新壓力、新資訊差、新人物反應、新後座、新位置變化或新未解狀態。

每次切場前須確認：

- 時間是否改變；
- 地點是否改變；
- 視角焦點是否改變；
- 人物下一步是否被改變；
- 讀者是否需要重新記住新的必記資訊。

若只是換地點補資訊、換人重述同一件事、或為了讓更多角色露臉而切場，應合併、略寫或刪除。

## 4. 章節開頭不得預設全知報告

章節開頭優先貼近一名人物、一個物件、一個變動點或一個不對勁的動作。背景、制度、群像反應與長線意義須隨人物行動逐步浮出。

適合的開頭切點包括：

- 名字已經被念過一次；
- 場次提前、顏色改了、門禁不開；
- 座位被佔、器材不在原位、終端通知延遲；
- 對手先到、某人缺席、醫療門外多了一個人；
- 公告流程本身失常。

若前三段已同時解釋時間、地點、制度、多人心理、長線意義與本章主題，須重寫開頭焦點。

## 5. 重複意象須有功能變化

櫻花、時計紋、影子、棋格、白線、公告牆、門、雨、糖、餅乾、終端、入場證與醫療門等意象或物件可反覆出現，但每次須承擔不同功能。

可承擔的功能包括：

- 能力徵象；
- 資訊差；
- 關係偏移；
- 身體或心理後座；
- 誤判與反制；
- 制度壓力；
- 情緒殘留；
- 下一場或下一章的具體麻煩。

不得只因漂亮、熟悉或方便收尾而重複同一意象。若同一符號連續多章只承擔同一種情緒，須降頻、換功能或改用普通動作直寫。

## 6. 句子朗讀感是清稿標準，不是裝飾

完稿清稿須通過朗讀感檢查。不同場面應有不同敘述密度：

| 場面 | 句子功能 |
| --- | --- |
| 戰鬥突發 | 短句、近鏡頭、身體反應優先。 |
| 戀愛猶豫 | 留白、停頓、沒說完、物件與站位承情緒。 |
| 群像公共場 | 多聲源、錯開對話、動作插入，避免排隊發言。 |
| 公告／裁定 | 簡短明確，不詩化、不替人物講情緒。 |
| 醫療／傷後日常 | 普通動作中帶不方便，不用旁白整理痛苦。 |
| 設定必要交代 | 嵌入人物正在處理的事，不獨立成報告。 |

若某段朗讀時像報告、口號、設定摘要、作者總結或漂亮句堆疊，須重寫為人物行動、對話或現場物件。

## 7. Scene 與 Summary 必須分流

承重事件必須寫成場景，低承重過程可摘要或跳過。

須寫成場景的內容包括：

- 正式勝負、受傷、醫療中止、認輸或失格；
- 關係偏移、錯過、拒絕、偏心或不成熟反應；
- 能力徵兆、能力失誤、反制與代價；
- 重大資訊差、誤判、隱瞞或支線下一台階；
- 角色正式作出的下一步選擇。

可摘要或跳過的內容包括：

- 無新變化的移動、等待、排隊；
- 已知資訊的重複說明；
- 無人物選擇的普通善後；
- 僅為交代場地而存在的過場。

不得把承重事件用一句旁白帶過，卻把低承重流程寫成完整場景。

## 8. 去模板清稿最低檢查

完稿前須檢查四項：

1. **聲線：** 遮掉名字仍能否分辨承重角色？
2. **轉場：** 每次切場是否帶來新壓力、新資訊或新後座？
3. **意象：** 重複物件或符號是否換了功能，而非只重複漂亮？
4. **朗讀：** 是否有段落像報告、流程、設定摘要或 AI 式漂亮總結？

任一答案為否，須重寫相關段落，不以補一句解釋通過。

# 八、場景因果節拍、Try／Fail 與情緒呼吸層

本層用於防止場景只是發生事情、展示資訊或完成流程。每個承重場景都必須讓人物帶著目標進場，遭遇阻力，得到有代價的結果，並因反應而改變下一步。

## 1. 承重場景須形成「目標 → 阻力 → 結果 → 反應 → 抉擇」

每個承重場景須具備基本因果節拍：

1. **目標：** 角色此刻要拿到、保住、避開、確認或隱瞞什麼。
2. **阻力：** 人、制度、場地、對手、傷勢、能力限制、情緒、資訊差或時間壓力如何阻礙。
3. **結果：** 成功但有代價、失敗且更糟，或取得答案但引出新問題。
4. **反應：** 角色的身體、情緒、話語、沉默或行動立即改變。
5. **抉擇：** 角色因此留下、離開、改場次、改說法、隱瞞、追問、放棄、誤判或承擔後座。

若場景結束後人物沒有新決定、資訊沒有新位置、壓力沒有升級或轉向，該場景應刪除、合併或重寫。

## 2. Try／Fail 結果不得只有成功或失敗

角色嘗試達成目標時，優先使用具有後續推力的結果：

- **Yes, but：** 成功了，但代價更高、被看見、受傷、排程變差、關係變僵或暴露弱點。
- **No, and：** 失敗了，而且失去時間、位置、名額、判斷優勢、同伴窗口或原本的退路。
- **No, but：** 失敗了，但取得一個資訊、看見一個破口、保住較小的東西或讓下一步更具體。

不得讓失敗只表示「沒成功」，也不得讓成功毫無副作用。正式競技、社團日常、戀愛靠近、醫療等待、觀戰與暗線調查皆可使用此原則。

## 3. 內心戲須貼身，不得替角色做總結

內心戲只寫視角人物當下能承受、能承認、能誤解或下意識避開的內容。不得把內心戲寫成作者替角色整理人格、主題或關係結論。

優先寫：

- 角色注意到什麼、漏看什麼、刻意不看什麼；
- 角色把情緒誤認成技術問題、時間問題、物件問題或別人的問題；
- 角色不願承認的念頭如何被動作替代；
- 角色下一步因此變慢、變硬、變短、變急或改方向。

少用「她明白了」「他終於知道」「兩人距離更近」「她其實很在意」等總結句。

## 4. 對話必須改變場面

每段承重對話至少要完成一項場面任務：

- 推動下一步；
- 改變關係距離；
- 隱瞞或暴露資訊；
- 轉移話題以逃避真意；
- 製造誤會或解除誤會的一部分；
- 釋放或提高張力；
- 迫使角色承認一個行動上的選擇，而不是口頭上的結論。

若一段對話只是在解釋讀者已知資訊，須刪減、改成動作、或讓對話造成新後果。

## 5. 高壓章節需要呼吸點，但呼吸點不得清空代價

高壓章節中可設置呼吸點，但呼吸點不是休息用空白，也不是把事件安慰掉。呼吸點應完成至少一項功能：

- 讓角色短暫恢復普通生活感；
- 讓關係以非說教方式偏移；
- 讓傷勢、失敗或名額壓力在日常中留下不便；
- 讓讀者看見角色仍有想守住的東西；
- 為下一個壓力點形成對比。

不得用笑點、吃飯、閒聊、甜場或善意照顧直接抹除勝負、傷勢、失敗與後座。

## 6. 幽默須來自人物與場面，不得硬插梗

幽默須來自角色本人、場面落差、地位變化、物件麻煩、資訊錯位、熟人默契或生活阻力，不得靠作者硬拗比喻、網路梗、固定口頭禪、怪句或破壞張力的吐槽。

嚴肅場面中的幽默只能釋放一小段壓力，不得取消事件代價。若笑點使傷勢、敗北、關係裂痕或危機後果像沒發生過，須刪除或重寫。

## 7. 每章須辨識主要承諾類型

每章起稿前須判定本章主要承諾類型，避免同時開太多方向卻不支付：

| 承諾類型 | 正文責任 |
| --- | --- |
| Milieu／場域 | 進入某場地、制度、外校文化或城市節點；章末須離開、改變對該空間的理解，或看見其新規則。 |
| Inquiry／問題 | 提出疑問、異常、資訊差或調查目標；章末須回答、部分回答，或使問題變得更具體。 |
| Character／人物 | 人物不滿、逃避、偏心、失衡或想要被壓住；章末須讓其情緒位置或下一步行動改變。 |
| Event／事件 | 某事件打破狀態；章末須使事件被處理、升級、轉向或留下可承接後果。 |

章末不滿足時，先檢查是否開錯承諾、開太多承諾，或把主要承諾藏在低承重流程後面。

## 8. 張力與釋放須形成節拍

章節內須安排張力與釋放的節拍。連續高壓後可有短暫釋放，但釋放點須留下新的問題、關係偏移、身體不便、資訊差或下一步壓力；連續低壓日常後須有微張力或變動點拉回故事。

不得全章同一壓力強度，也不得全章只靠結尾突然拉高張力。戰鬥、戀愛、群像日常、醫療等待、公告與社團場景皆應有不同的呼吸節拍。

# 九、伏筆觸發、反應場景、敘事距離與修稿防迴圈層

本層用於防止長篇候選正文出現「伏筆只剩氣氛」「重大事件後沒有反應」「旁白替角色總結」「越修越像規則文件」等問題。正文須讓伏筆、反應、敘事距離與修稿停止條件互相配合，而非只堆疊正確要素。

## 1. 伏筆須具備「伏筆 → 觸發 → 支付」結構

伏筆不得只保存為「可疑痕跡」。每個承重伏筆至少應能對應以下三段：

1. **伏筆：** 正文中出現可被讀者記住的異常、物件、話語、能力徵象、場地反應或人物選擇。
2. **觸發：** 未來某個條件使該伏筆重新變得必要，例如同類能力再次出現、同一人物再遇相似選擇、場地紀錄被調出、對手利用已暴露情報、某人說出相反證詞。
3. **支付：** 伏筆造成具體結果，例如勝負改變、資訊差縮小、人物誤判、關係偏移、危機升級或設定真相局部成立。

若伏筆沒有可想像的觸發條件，不應反覆出現；若伏筆尚未支付，正文不得把它寫成已成立答案。低濃度暗線可以只露痕跡，但痕跡必須落在可被再次調用的物件、紀錄、話語、場地或人物行動上。

## 2. 重大事件後須有反應場景或反應節拍

重大勝負、受傷、公開失敗、能力徵兆、資訊揭露、關係偏移與伏筆觸發後，不得只寫「大家震驚」「她沉默」「他明白了」。正文須至少呈現以下一項，承重事件則應盡量形成多層反應：

- **身體反應：** 疼痛、疲勞、手抖、呼吸變短、步伐改變、不能再做某動作或武裝回收變慢。
- **認知反應：** 角色誤解、拒絕理解、抓錯重點、只承認一半、把情緒轉成技術問題或避開真正原因。
- **行動反應：** 改排程、刪訊息、避開某人、追問、隱瞞、留下、離開、申請複查或改變下一步。
- **關係反應：** 某人沒接住、幫錯忙、越界、退開、偏心、多做一步或刻意少做一步。

若角色受傷、輸掉、獲勝、暴露能力、錯過場次、聽見秘密或看見伏筆後，下一步仍和事件前相同，該事件後座不足，須補寫或重構。

## 3. 敘事距離須依場面功能調整

敘事距離不是固定遠近。正文須依場面功能選擇貼近或拉遠：

- 戰鬥、受傷、失敗、心動、羞恥、誤判與關係裂縫時，優先貼近視角人物的感官、動作、錯認與不願承認。
- 制度、公告、場地轉換、低承重時間流逝、多人同時移動或背景狀態，可拉遠摘要。
- 不得在貼身場面突然跳出作者總結人物心理，也不得在需要清楚資訊時故意過度朦朧。

若一句話能被改成視角人物當下看見、聽見、碰到、避開或做出的動作，優先改成動作。少用「她其實已經明白」「他終於理解」「兩人的距離更近」等外部判定句。

## 4. 修稿須有停止條件，不得越修越像規則文件

內部修稿只處理會影響故事成立、人物真實、因果清楚、語言自然的問題；不得為了讓每條規則都顯性出現，而把正文改成教科書、流程報告或設定展示。

修稿停止條件：

1. 本章主事件已支付具體變化。
2. 主要人物行動符合本人目的、限制與當下情緒。
3. 勝負／能力／關係／伏筆因果可追溯。
4. 章末留下具體後座或未解狀態。
5. 朗讀時不像報告、設定摘要、口號或漂亮句堆疊。

若以上皆成立，不再為了增加象徵、金句、伏筆、解釋或整齊結論而加段落。

## 5. 章節鉤子須分級使用

章節需要讓讀者想看下一章，但不必每章都用大危機或爆點收尾。鉤子須服從篇章濃度與本章已支付事件。

| 鉤子等級 | 合法用途 |
| --- | --- |
| 小鉤子 | 未回覆訊息、改登記時間、傷勢影響下一場、某人避開問題、物件位置不對。 |
| 中鉤子 | 新對手看穿弱點、場次變動、能力徵兆不穩、醫療複查、場地紀錄異常。 |
| 大鉤子 | 代表名額重大變動、暗線行動、重大傷勢、危機爆發或篇章節點。 |

章末鉤子必須來自本章已支付事件，不得空降新危機或只用抽象不安硬拉期待。

## 6. 人物弧線不得每章都成熟

角色受壓後不必每章都成長、理解或和解。有效推進可以是：

- 更嘴硬；
- 更逃避；
- 更不甘；
- 更偏心；
- 更沉默；
- 只改變一個小動作；
- 表面沒變，但下一步安排已經偏離原本軌道。

人物弧線的有效推進，不等於人格變好，而是人物的選擇、限制、弱點、關係位置或自我誤解被讀者看得更清楚。

## 7. 摘要段落也須有方向

低承重移動、等待、排隊、收拾、換場、通訊背景或重複善後可摘要，但摘要不是無目的省略。摘要須交代至少一項：

- 時間推進；
- 位置改變；
- 資訊差保留；
- 人物狀態變化；
- 事件後座沒有被清除；
- 下一個場景為何能成立。

不得用摘要偷渡承重事件，也不得把摘要寫成流水帳。勝負、傷勢、關係偏移、能力徵兆、重大資訊差與正式選擇，仍須寫成場景。


# 十、刺激反應、感官焦點與場景顯著性層

本層用於處理逐段可讀性，防止正文先總結心理、再補事件原因，或讓低承重流程佔據完整場景。角色必須先被外部刺激打到，再由身體、判斷、行動與語言形成反應；場景只保留真正改變人物、資訊、關係、能力、排程或伏筆位置的部分。

## 1. 段落須遵守「刺激 → 反應」自然順序

承重段落原則上依下列順序展開：

> 外部刺激 → 非自願感覺／身體反射 → 動作調整 → 判斷／情緒 → 語言／決定

外部刺激可以是對手動作、場地變化、公告推播、旁人一句話、醫療燈號、物件位置、靈力異常、視線、沉默或某個人沒有做本來會做的事。

正文應先讓讀者看見事情如何打到角色，再寫角色如何被迫反應。不得先寫「她很震驚」「他明白了」「她突然不安」，再回頭補原因；除非刻意製造資訊延遲，否則反應順序錯亂會削弱沉浸感。

## 2. 非自願反應優先於完整心理結論

角色被刺激擊中後，先出現的往往不是完整想法，而是身體與動作：

- 受傷時先是呼吸斷掉、手指鬆開、腳步偏掉、視線縮窄；
- 被關心時先是停頓、避開、拿錯物件、把話題推回技術細節；
- 被逼問時先是沉默、改站位、看向出口、整理不需要整理的東西；
- 看見關係偏移時先是回覆變短、終端停在未送出畫面、原本要走的腳步沒有動。

不得讓角色第一時間說出完整心理結論。內心可以存在，但應貼近角色當下能承認、誤認或逃避的那一小段。

## 3. 感官描寫須依視角人物與場面功能選擇

感官不是裝飾，不必五感全塞。每個感官細節都應揭露角色狀態、場面壓力、世界生活感或後續資訊。

| 場面 | 優先感官與描寫方向 |
| --- | --- |
| 戰鬥／正式競技 | 觸覺、重心、聲音、呼吸、武裝受力、視線死角。 |
| 戀愛／慢熱靠近 | 距離、停頓、手上物件、聲音變化、未完成動作。 |
| 醫療／傷後等待 | 燈光、消毒氣味、儀器音、布料摩擦、不能進門的距離。 |
| 觀戰／公告 | 人群聲浪、結界震動、視線遮擋、推播延遲、名字被看見的壓力。 |
| 社團／日常 | 桌面物件、登記簿、椅子位置、飲料、終端、誰順手或不順手。 |

同一場景的感官焦點應與視角人物有關。千夜可能先聽見節拍，貓狼可能先看光源與影子，御先可能先注意棋格與座位，九逃可能先看傷勢與呼吸；不得用同一套感官描寫套所有角色。

## 4. 承重角色須具備能動性

承重角色不能只是在心裡理解、旁觀、被照顧、被說服或等待別人決定。角色的行動可以很小，但必須改變場面、資訊差、關係距離或下一步。

有效能動性包括：

- 改登記時間、提前離席、留下、繞路或拒絕同行；
- 沒有回訊息、刪掉訊息、只傳紀錄不傳安慰；
- 擋住一句追問、把話題推回規則、假裝只是在處理物件；
- 把觀戰紀錄發出去、把傷勢瞞下來、把座位讓出或不讓；
- 在正式競技中改策、放棄原本優勢、逼對手交出下一步。

若角色整章只在理解、旁觀、被保護或被劇情推動，須重寫其主動行動。

## 5. 對手與反派須主動反制

承重對手與反派不是障礙物，而是會回合外行動的人。正式競技中，對手至少應有一次主動迫使主角改策的行動；危機或暗線中，反派行動須留下可追查的具體手法。

對手可主動利用：

- 已公開情報、前測紀錄、觀戰所得或主角習慣；
- 場地白線、結界邊界、光源、死角、醫療中止線；
- 正式規則、裁定節奏、觀眾壓力或時間限制；
- 主角的情緒弱點、熟人顧忌、偏心、逞強或不願暴露的能力限制。

不得只用抽象壓迫感、資料異常、制度流程或旁白說明代表敵意。反派與對手的行動必須落到人、物件、場地、排程、傷勢、資訊差或可追查後果上。

## 6. 場景顯著性決定篇幅

完整場景應留給真正改變故事位置的轉折點：勝負、受傷、能力徵兆、關係偏移、資訊差改變、支線台階、伏筆觸發、對手主動反制或人物主動選擇。

低顯著場景如普通移動、等待、重複登記、無新資訊的閒聊、無新變化的善後，應摘要、合併或刪除。

若一場戲沒有改變人物、資訊、排程、關係、能力、伏筆或世界後座，不應佔用完整場景篇幅。

## 7. 章節中段不得空轉

章節中段不能只延長等待、重複壓力或把開頭已知資訊換人說一遍。中段至少須完成一項推進：

- 對手策略揭露；
- 能力限制暴露；
- 場地條件改變；
- 人物誤判或改策；
- 關係距離變化；
- 資訊差擴大或縮小；
- 支線台階形成；
- 伏筆被觸發但尚未支付。

若中段只是在重申開頭已知壓力，須壓縮、合併或重構。

## 8. 段落資訊須節制，避免同拍過載

同一段落內不要同時塞入太多功能：動作、能力說明、內心分析、世界設定、旁人反應、伏筆暗示與漂亮意象若全部擠在一起，會讓讀者失去焦點。

每段優先承擔一至兩項主要功能：

- 動作推進；
- 身體反應；
- 資訊揭露；
- 關係偏移；
- 能力條件；
- 場地壓力；
- 情緒潛台詞。

若段落同時在解釋設定、推進戰鬥、描寫情緒與埋伏筆，須拆段、延後或改以物件／動作承載。讀者先看懂當下，再承接長線。

# 十一、讀者定位、謎題清晰與沉浸感層

本層用於控制長篇多角色、多能力、多場景推進時的讀者理解負荷。正文可以保留未知、資訊差與暗線懸念，但不得讓讀者迷失當下應關心的目標、阻力與後果。

## 1. 神秘與混亂必須分開

正文可保留謎團，但不得製造混亂。讀者可以不知道暗線真相、能力全貌、組織目的、家族伏筆或角色隱瞞內容；但必須知道當下場景的表面目標、人物想要什麼、阻力在哪裡、結果影響誰。

若讀者不知道該關心哪一件事，該謎團不是懸念，而是資訊混亂。能力、制度、伏筆與新角色可以同場存在，但必須有一條清楚的讀者問題把它們串起來。

## 2. 切場後須迅速重新定位讀者

每次切場、跳接、換視角或時間略過後，正文須用最少文字完成讀者定位：目前時間、地點、視角焦點、人物正在做什麼、與上一場的因果關係。

定位不等於背景說明；可用物件、聲音、門禁、座位、傷處、終端推播、場地反應、排程顏色或人物正在處理的東西完成。

若讀者需要重讀才知道場景換到哪裡、誰在場、這場和上一場有什麼關係，轉場定位不足，須重寫。

## 3. 章內資訊須有新鮮度曲線

章內段落須避免連續重複同一類資訊。壓力、設定、情緒、能力說明、觀戰反應、日常動作與伏筆痕跡須交替推進。

若連續數段都只在表達同一件事，例如「正式競技很危險」「某人很尷尬」「兩人很在意」「暗線很神秘」，須刪減或改成新的行動、資訊差、反制、後座、選擇或場景波峰。

每一段至少應比上一段多一個新功能：讓人物更接近目標、讓阻力更具體、讓資訊差改變、讓情緒位置偏移、讓對手改策、讓世界摩擦落地，或讓章末後座更可承接。

## 4. 承重場景須具備沉浸三入口

每個承重場景須同時提供三種沉浸入口：

1. **注意力入口：** 讀者知道視角人物正在注意什麼、避開什麼或誤判什麼。
2. **具體意象入口：** 場景中至少有一個可感物件、聲音、動作、傷處、場地反應或能力徵象。
3. **情緒壓力入口：** 讀者能感覺人物的當下壓力、慾望、抗拒、不甘、偏心或逃避。

若場景只有設定資訊，缺少人物注意力；只有漂亮畫面，缺少行動壓力；只有情緒旁白，缺少具體物件，皆須重寫。

## 5. 感官交錯可用，但必須有來源與功能

感官交錯可用於能力、情緒與場地質感，但必須有角色感知來源。時間、影、重力、光紋、治療、結界、名字、記憶、封存等能力，可透過聲音、觸感、溫度、壓力、視線延遲、呼吸變化、疼痛或空氣變化呈現。

不得為了文采堆疊無因感官。感官描寫應服務能力成立、人物壓力、場地定位、資訊差或後續判斷。

## 6. 回顧只補本章必要接點

長篇回顧不得重述完整設定或前情。再次提到舊事件、舊傷勢、舊對局、舊關係、舊伏筆時，優先用一個物件、一句對話、一個動作、一個場地反應或一個當前後果喚回讀者記憶。

若回顧內容不會改變本章行動、資訊差、情緒位置或後座，應刪減。不得因擔心讀者忘記而把正文寫成前情摘要。

## 7. 每個承重場景須帶出讀者問題

每個承重場景開始時，應讓讀者形成一個可追蹤問題：她能不能趕上？他會不會說破？這招會不會被看穿？門會不會開？對手到底在等什麼？某人為什麼沒回訊息？

場景結束時須回答、部分回答，或把問題轉成更具體的新問題。不得讓場景只提供資訊而不製造讀者問題。

## 8. 多資訊場面須先收斂焦點

同一場面若同時包含新角色、能力、制度、伏筆、關係情緒與世界細節，須先決定讀者此刻最該關心的一件事。其餘資訊只能作背景、質感、反應或後續鉤子，不得同時搶主焦點。

每個承重場景須讓讀者清楚知道「當下該關心什麼」。若場景同時拋出太多名詞、伏筆、情緒、能力與制度，卻沒有清楚讀者問題，須收斂焦點。

# 十二、連載回顧、支線編織與讀者負荷收束層

本層用於長篇群像連載之持續可讀性管理。正文可以擁有大量角色、支線、能力、場地與伏筆，但每一章都必須讓讀者知道誰重新進場、哪條線被推進、本章哪些資訊需要記住，以及章末既支付了什麼、又留下什麼下一步動力。

## 1. 角色重新登場須有辨識錨點

角色隔章或久未出場後重新登場時，須提供一個低負擔辨識錨點，使讀者能自然接回此人，不必重讀角色資料。

可用辨識錨點包括：

- 角色正在做的事；
- 所屬場地或生活動線；
- 手上物件、入場證、登記簿、外套、終端或傷處；
- 聲線特徵、處理尷尬的方式、反問方式或沉默方式；
- 能力徵象、武裝媒介或戰鬥習慣；
- 上一次仍有效的後座；
- 其與本章主事件的具體關係。

不得以長段履歷式介紹重述角色背景；角色重新登場應像生活中再次遇見某人，而不是打開資料卡。

## 2. 回顧只接當前後果

回顧不是摘要前情，而是接回本章正在使用的後果。舊傷、舊勝負、舊關係、舊伏筆、舊場地或舊能力徵象再次出現時，應以當前仍有效的物件、排程、身體限制、對話避讓、場地反應或資訊差喚回。

優先使用：

- 被摺皺或被壓住的入場通知；
- 醫療貼、複查時間、不能用力的手；
- 沒有回覆的訊息；
- 某人刻意避開的稱呼；
- 被改掉的登記時間；
- 場地上仍未撤下的標示；
- 觀戰紀錄、公告截圖或裁定摘要。

不得每章重講完整設定、完整前情或完整人物關係；若回顧內容不會改變本章行動，應刪減。

## 3. 多支線同章出現時須互相照應

多支線同章出現時，至少一條支線須與本章主戲形成對照、鏡像或反差，而不是平行堆疊。

可形成照應的方式包括：

- 有人正面爭取，有人選擇避開；
- 有人因勝負被看見，有人因缺席被注意；
- 有人靠近，有人退開；
- 有人依靠規則，有人鑽制度縫隙；
- 有人承認受傷，有人隱瞞負荷；
- 有人因公開結果改變位置，有人因私下選擇改變距離。

支線不得只是補設定、補角色出場或增加篇幅；若支線不能照出主戲的新面向，應延後、刪減或改成背景動線。

## 4. 承重對話須有地位或控制權變化

群像對話不得只是平均分配資訊。承重對話中，須能看見誰取得主導、誰被逼退、誰迴避、誰轉移話題、誰用沉默擋住追問，或誰因一句話改變下一步。

對話中的地位變化可以很小：

- 問句被反問擋回；
- 原本催促的人被迫停下；
- 某人用技術細節避開情緒；
- 某人沒有回答，卻把物件推過去；
- 一個人結束話題，另一個人只好改動作；
- 旁人只聽到半句，造成後續誤會。

若一段對話只是大家輪流說明資訊，沒有任何人位置、話題、關係距離或下一步改變，須刪減或改成行動。

## 5. 本章新增資訊須分級

每章新增資訊須明確分成 A／B／C 三級，避免讀者負荷過重。

| 等級 | 定位 | 例子 | 寫法 |
| --- | --- | --- | --- |
| A 級 | 後續必須承接之本章必記資訊 | 勝負、傷勢、能力徵兆、場次變動、關係偏移、正式支線台階 | 必須清楚落地，不可藏在背景裡。 |
| B 級 | 篇章內可能回收之資訊 | 臨時對手特徵、場地規則、可疑紀錄、外校文化、一次性制度限制 | 可具體但不反覆強調。 |
| C 級 | 世界質感或氣氛資訊 | 店名、路人、環境細節、低風險生活科技 | 服務場景即可，不要求讀者記住。 |

正文不得把 C 級資訊寫得像 A 級，也不得讓 A 級資訊只作背景質感處理。若一章同時出現過多 A 級資訊，須刪減、合併或延後。

## 6. 摘要須分清三種用途

摘要可用，但須知道用途，不得偷渡主戲。

| 摘要類型 | 用途 | 防錯 |
| --- | --- | --- |
| 時間摘要 | 跳過低承重移動、等待、排隊、重複交代。 | 不得把正式勝負、能力突破或重大關係轉折藏進摘要。 |
| 狀態摘要 | 交代人物、傷勢、排程、關係或場地的目前位置。 | 必須指向本章行動，不得成為設定回顧。 |
| 回顧摘要 | 喚回本章必要前情。 | 只補必要接點，不重述完整前情。 |

勝負、受傷、能力徵兆、重大資訊差、關係偏移、伏筆觸發與正式選擇必須寫成場景；摘要只能橋接，不能代替支付。

## 7. 章末須同時有本章支付與下一步動力

章末須同時保有「本章支付」與「下一步動力」。

- 本章支付回答讀者：這一章為什麼值得存在？
- 下一步動力回答讀者：之後還有什麼會改變？

本章支付可以是：

- 一場勝負或中止；
- 一個傷勢、負荷或複查安排；
- 一段關係偏移；
- 一項資訊差改變；
- 一個支線台階；
- 一個伏筆被觸發；
- 一個人物主動選擇。

下一步動力可以是：

- 下一場或下一次檢查；
- 未回覆訊息；
- 對手留下的觀察；
- 場地紀錄或物件異常；
- 尚未說出口的決定；
- 某人因本章結果改變動線。

不得只留下鉤子而沒有本章支付，否則像預告；也不得支付完後完全沒有下一步，否則像短場景而非連載章節。

# 十三、回呼支付、敘事壓縮與規則降噪層

本層用於避免寫作卡規則過度外顯，使正文在具備承重、因果、伏筆、戰鬥、能力、戀愛、群像與連載後座的同時，仍保持小說的乾淨、流動與閱讀回報感。

## 1. 回呼不是重複，而是前文在新位置產生新意義

回呼是讓讀者感覺「前面不是白看」的敘事回報，不是把同一物件、同一句話或同一場地重複展示。

重複物件、台詞、場地或動作再次出現時，必須因人物位置、關係、勝負、傷勢、能力狀態或資訊差改變而產生新意義。

可作回呼的元素包含：

- 糖、餅乾、飲料、外套、入場通知、觀戰證、終端訊息；
- 公告牆、白線、醫療門、觀戰區、影術棋盤桌、訓練館入口；
- 刀鞘、時印、時計紋、影子、棋格、結界痕、武裝擦痕；
- 某人曾經說過但這次說不出口的話；
- 某人曾經做過但這次少做、多做或反過來做的動作。

不得只因熟悉、漂亮或可愛而重複；若再次出現沒有新功能，應刪除、淡寫或換成當下更有效的動作。

## 2. 敘事壓縮：每一筆都要有功能

完稿前須執行敘事壓縮。正文不追求把所有規則、設定、情緒與場面都寫滿，而是讓留下的每一筆至少服務下列一項：

- 人物行動；
- 場景定位；
- 勝負／能力因果；
- 關係後座；
- 伏筆觸發或支付；
- 情緒壓力；
- 世界生活感；
- 讀者問題或下一步動力。

不能服務上述功能者，即使句子漂亮、設定有趣、意象熟悉，也應刪除、合併或降為摘要。

正文不得為了展示自己有做到「微張力、群像、後座、伏筆、能力條件、戰鬥受力」而把同一段寫得過滿。讀者只需要看見人物正在現場生活與承擔結果，不需要看見規則清單。

## 3. 摘要灰泥原則

摘要與過渡只作灰泥，不作主戲。它們負責推進時間、重新定位、銜接低承重資訊、保留當前後座或把讀者帶到下一個有效場景。

摘要可以處理：

- 低承重移動；
- 普通等待；
- 無新資訊的排隊與登記；
- 重複善後；
- 已知流程的時間推進；
- 僅需交代目前位置的簡短回顧。

摘要不得偷渡：

- 正式勝負；
- 重大傷勢；
- 能力突破或穩定新用法；
- 重大關係轉折；
- 伏筆支付；
- 正式選擇與不可逆後果。

若摘要段比場景更像主戲，須改寫成場景；若場景只是灰泥，須壓縮成摘要。

## 4. 承諾債管理

每章不得無限制新增承諾。新增對局、能力徵兆、暗線痕跡、關係偏移、新角色、新場地、新制度或新問題時，須同時確認本章是否支付或推進至少一項既有承諾。

承諾債過高的徵兆：

- 連續數章都只增加新名字、新能力、新異常或新關係問題；
- 舊的傷勢、勝負、名單、能力徵兆或伏筆沒有被提及、觸發或支付；
- 章末靠新鉤子維持期待，但本章沒有回答任何既有問題；
- 讀者需要記住很多「以後會重要」的東西，卻看不到當下回報。

若承諾債過高，須刪減新鉤子，或把本章改為支付舊線、觸發舊伏筆、落地舊後座。

## 5. 篇幅重量匹配

篇幅須符合事件重量。正式勝負、傷勢、能力徵兆、關係偏移、資訊差改變、伏筆觸發、支線台階與章末後座，應取得足夠場景篇幅。

普通移動、排隊、重複公告、無新資訊閒聊、低承重善後與已知流程，應摘要、合併或刪除。

若小事寫太滿、大事寫太薄，讀者會誤判本章重點；須重排場景、剪掉灰泥或補足承重場面。

## 6. 連貫缺口檢查

正文可保留謎團，但不得留下非故意的連貫缺口。下列事項須可追蹤：

- 時間推進；
- 場地位置；
- 人物出入；
- 傷勢與醫療狀態；
- 武裝是否具現；
- 場次是否已開始、結束、中止或改排；
- 資訊由誰知道、誰不知道、誰誤以為知道；
- 重要物件目前在哪裡；
- 章末後座是否接得上下一步。

真正的暗線可以不解釋，但基本行動連續性不得讓讀者自行補洞。

## 7. 規則降噪：寫作卡只能在後台

寫作卡只作後台，不得成為正文表面。正文不需要顯性展示所有規則；只要故事自然成立即可。

完稿前須刪除任何看起來像在展示規則、補齊清單、證明有後座、硬塞伏筆或向讀者說明「這裡有意義」的段落。

正文優先序永遠是：

> 人物當下想做什麼 → 做了什麼 → 因此失去、改變或留下什麼。

規則只在背後防止失真，不應變成正文的語氣、結構或自我說明。

# 十四、人物、群像、配角與對手

## 1. 承重人物必須有自己的目的

人物不是等待劇情觸發的功能位置。出場時應帶著自己的利益、時間、面子、責任、習慣或不願被看見的弱點。

人物可以：

- 誤會、煩躁、答錯重點或不想安慰人；
- 先保住自己的機會、時間、名額或尊嚴；
- 嘴上說沒事，行動卻已經改變；
- 在應該說漂亮話時說不出口；
- 明明在意，卻不願意先承認。

## 2. 群像不等於所有人一起關心同一件事

三人以上同場時，應允許不同生活軌道同時存在：

- 有人正在趕自己的事；
- 有人只聽到後半句；
- 有人不想捲入情緒；
- 有人先顧自己的東西或排程；
- 有人的選擇反而讓現場更麻煩。

主角群地位平等，不必每章全員出場，也不得長期只讓固定角色壟斷推進權。

## 3. 配角與對手必須是活人

配角與新角色不得只負責遞資訊、見證主角、評論關係或善後。

正式競技中的對手必須具備：

- 自己想贏或不願失去的理由；
- 清楚可見的強點與打法；
- 看過情報後調整策略的能力；
- 敗北、受傷或中止後屬於本人的反應。

不能為了讓預定勝者精彩，而讓對手突然失去基本判斷力。

# 十五、對話、旁白與日常語言

## 1. 對話先處理眼前事

人物說話首先應服務當下目的：確認、催促、推責、拒絕、遮掩、試探、爭方法、處理結果或故意轉開話題。

避免：

- 角色替作者講出本章結論；
- 對話排隊解釋設定；
- 剛受傷、失敗或失位就說出完整人生體悟；
- 旁人精準替讀者宣布兩人關係進展。

## 2. 情緒落在小動作與選擇上

壓力、不甘心、靠近或疏遠，優先寫成：

- 回話變慢或變短；
- 少做了一個本來會做的動作；
- 明明能順手幫忙，卻沒有幫；
- 一直追問技術細節，不肯談感受；
- 願意承擔對方造成的不方便；
- 繼續同行，卻已經不是原本的相處方式。

## 3. 語言自然優先

- 普通行動直接寫清楚。
- 公告、裁定與系統可正式，人物口語不得被制度語氣污染。
- 熟人對話不反覆講完整專名。
- 新術語若非必要，不另造名稱。
- 不以漂亮比喻、流程語或旁白總結取代現場。

# 十六、物件、日常後續與章末

## 1. 物件先是生活中的東西

通知、入場證、終端、外套、座位、傷處、飲料、繃帶與器材，必須先因人物真的要用、弄丟、帶不走、拿錯、弄壞或受到影響而存在。

物件可以自然承載人物狀態，但不得從頭到尾反覆替同一情緒封義。

## 2. 日常不得擦掉主戲代價

競技、受傷或摩擦之後，人物仍可吃飯、回教室、傳訊息、同行或做社團事務；但日常必須承接前因：

- 原定安排因此被改動；
- 某人不能再照原本方式行動；
- 同行是因責任、同路或尚未處理完的麻煩；
- 笑點不負責把傷勢、敗北或關係失衡沖掉。

## 3. 章末停在仍存在的後果上

適合收尾的位置：

- 某人做出不同於先前的下一步；
- 某人沒有回答；
- 某場勝負造成的新安排落到人物身上；
- 傷勢、麻煩或關係不順仍未解決；
- 某人仍得帶著結果繼續生活。

章末不必補結論、漂亮句子或旁觀評語。

# 十七、戀愛與日常場面描寫層

戀愛與日常不得寫成休息場、甜場、資訊整理場或主題收束場。每個日常場景都必須先成立人物原本要做的事，再讓另一人的存在、事件後果、社團安排、場次時間、門禁、通訊、醫療規則或生活物件造成偏移。

情感推進不靠旁白宣布、不靠旁人標註，也不靠一次照顧、同行、對局或受傷直接成立關係成果；它應從人物的選擇成本、生活阻力與沒說出口的反應裡長出來。

## 1. 日常先有各自的生活軌道

日常場景開始前，每名承重人物都應有自己原本要做的事：

- 趕場次、排隊、登記、收器材、佔座、買飲料、回訊息、等靈軌；
- 想避開某人、想看比賽卻不承認、想幫忙但不想被看見；
- 被自己的社團、班級、家庭、醫療複查或正式選拔排程卡住。

不得讓角色只因主角需要聊天、需要被安慰、需要補資訊而集合。群像日常中，同一場景裡的人可以各自趕時間、保住面子、處理物件、誤會對方或只聽見部分資訊。

## 2. 戀愛推進以選擇成本為核心

戀愛與曖昧不以「心動」或「變近了」作為直接成果，而以人物為對方付出的微小成本成立：

- 多等一段時間；
- 改掉原本安排；
- 承擔一點不方便；
- 替對方留位置、留資訊、留物件；
- 不說安慰話，卻做了實際有用的事；
- 想靠近，卻因自尊、害怕或不成熟而用錯方式。

一次同行、幫忙、對局、照顧傷勢或共同看見同一件事，不得直接推成重大關係成果；它最多只能造成下一次相處的距離、語氣、座位、選擇或不自在略有變化。

## 3. 情緒優先藏在非語言反應裡

情緒應優先透過視線、手勢、腳步、聲音、停頓、座位距離、物件位置、訊息刪改與未完成動作呈現。

可寫：

- 手停在終端回覆欄上，打完又刪掉；
- 明明可以先走，卻慢了一步；
- 把飲料或紀錄放到對方容易拿的位置；
- 想問傷勢，出口卻變成技術細節；
- 少做了一個平常會做的動作；
- 回答變短，或故意把話題拉回正事。

少用或避免：

- 「她很在意」；
- 「他心動了」；
- 「兩人的距離變近了」；
- 「她終於理解自己的感情」；
- 由旁人精準標註兩人關係進展。

## 4. 青春關係允許不成熟與不和解

角色可以幫錯忙、說錯話、逃避、嘴硬、裝沒事、只問技術、不會安慰、沒有接住對方，甚至暫時不和解。

受傷、失敗、失位或公開出糗後，不得讓角色立刻說出完整體悟，也不得讓另一人用正確溫柔的話一次解開問題。關係中的不成熟、尷尬、偏心、怒氣與誤解可以留下，並在後續生活裡繼續影響座位、訊息、同行與選擇。

## 5. 生活制度與場景要參與關係變化

本作的戀愛與日常必須活在現代高科技都市與靈力科技校園裡。場次通知、觀戰區、醫療門禁、社團登記、終端推播、靈軌時間、公告更新、器材借還、生活靈紋與城市設施，都可成為靠近、錯過、誤會或暴露偏心的原因。

不得把戀愛場面固定抽離到空教室、屋頂、黃昏談心或純兩人甜場。真正有效的靠近，可以發生在排隊、收桌、醫療區外、場館出口、販賣機前、靈軌月台、社團值班與比賽後混亂的公共廳。

## 6. 戀愛與戰鬥後座必須互相影響

正式競技、傷勢、勝負與名額壓力會改變人物的日常與關係；戀愛或友情也會影響角色在競技後如何靠近、退開、沉默或說錯話。

可成立的方向包括：

- 戰鬥暴露某人的逞強，讓另一人之後無法照原本方式對待她；
- 受傷製造靠近機會，但醫療規則、本人自尊或旁人目光讓靠近不順利；
- 勝利讓某人被看見，也讓另一人意識到自己站得太遠；
- 失敗後不是被安慰，而是被對方用不溫柔但有用的方式指出問題；
- 因關係在意而做出的戰鬥選擇，必須留下勝負、傷勢、排程或心理後座。

日常不能沖淡戰鬥代價；戀愛不能抹平失敗；受傷照顧不能自動成為關係答案。

# 十八、正式競技與能力戰成稿規則

## 1. 先依最新版引擎辨識競技層級

| 場面類型 | 顯現與勝負方式 | 正文重點 |
| --- | --- | --- |
| 自由練習／選拔前測／試位測試 | 依最新版引擎可採投影顯現、得分、奪標、入圈、回合或測試格式。 | 重點為暴露打法、限制與參考結果，不得冒充正式資格勝負。 |
| 校內正式選拔／正式交流賽／全國代表競技 | 依最新版引擎指定之正式競技規格處理；未另指定為安全交流演練時，採本體顯現、單局決勝。 | 須支付具名對戰、真實攻防、勝負、傷勢與後座；不得寫回安全得分制。 |
| 真實危機介入 | 依正式危機、能力與場面因果成立。 | 救援、傷害、失能與死亡風險均須承擔後續。 |

## 2. 正式競技不得安全化或匿名化

正式競技作為本章主戲時，正文必須寫清楚：

- 誰在對戰；
- 雙方為何想贏；
- 哪一方先採取什麼策略；
- 對手如何判斷與修正；
- 哪一個空檔、誤判、限制或代價使結果成立；
- 誰贏、誰輸、誰受傷、誰被中止；
- 這個結果如何影響下一步。

不得無故事理由地：

- 只寫「左側那人／右側那人」；
- 只讓旁觀者理解正式競技很危險；
- 刻意避開單場結果；
- 把傷害固定縮成擦傷、手麻或檢查無事；
- 讓正式對局看起來仍只是安全測試。

## 3. 勝負因果必須可回溯

若本章以競技或能力戰承重，讀者必須能追溯：

> 前置資訊與限制 → 人物選擇 → 對手反應或誤判 → 形成空檔／代價 → 勝負或中止 → 身體與關係後座

場面中依需要寫出：

- 場地、位置與距離；
- 武裝具現與能力作用；
- 出手前的判斷與前置動作；
- 對手如何應對；
- 傷勢與失能如何實際成立；
- 裁定或醫療為何介入；
- 勝負後誰失去什麼或必須改變什麼。

## 4. 關鍵解法必須提前成立

決勝所依賴的規則、能力、場地條件、人物習慣或反制判斷，必須在結果前已有自然依據。

不得：

- 臨時新增能力完成逆轉；
- 事後才補出從未建立的規則；
- 讓對手忘記已看過的招法；
- 以多個未支付巧合堆出勝利。

## 5. 熟人對戰必須雙向成立

熟人對戰中的「熟悉」必須落在具體可見之處：

- 慣用步法、偏手、回收方式；
- 逞強時會守住或放棄的位置；
- 習慣性的假動作；
- 前次交鋒暴露過的弱點；
- 熟悉造成的正確預判或錯誤自信。

雙方都能利用熟悉，也都可能因此判錯；不得只有勝者單方面讀懂對手。

# 十九、戰鬥場面描寫層

戰鬥場面不得只寫招式名稱與結果，必須讓讀者看見人物如何在距離、重心、視線、武裝軌跡、能力條件、對手反應與身體負荷之間作選擇。

## 1. 戰鬥先有目標與進度

每場承重戰鬥開始前，正文須讓讀者知道至少一項具體目標：

- 角色要搶到、守住、避開或確認什麼；
- 這場勝負會影響名額、排程、身體、關係或公開位置中的哪一項；
- 場面推進時，讀者能感覺角色離勝利、失敗、受傷、中止或失位越來越近。

不得只寫「雙方交鋒數回合」或「最後抓到空檔」；空檔必須由前面的位置、壓力、誤判或代價自然累積出來。

## 2. 動作必須有身體與重量

每次承重交鋒至少應讓讀者理解：

- 人物腳步、重心、手腕、肩背或呼吸如何承受動作；
- 武裝撞擊、格擋、落空、擦過或被壓制時，力量如何傳回身體；
- 受傷、失衡、靈力紊亂、視線受阻或疲勞如何影響下一步；
- 角色不是憑意志瞬間重置狀態，而是帶著前一拍的代價繼續打。

不得只寫「他閃開」「她擋下」「光芒炸開」「影子纏住」而不交代距離、動作與受力。

## 3. 能力描寫須有可見成立順序

能力戰應依場面需要呈現：

> 武裝具現 → 靈力流向 → 可見徵象 → 作用範圍 → 對手如何察覺 → 反制或誤判 → 實際效果 → 負荷與後座

時間、影子、絲線、星紋、重力、光紋、結界、治療、生命靈流、通行權限與場域能力，都必須讓讀者看見至少一個可感媒介或判斷線索。能力效果不得只靠抽象詞成立。

## 4. 攻防狀態不得無代價歸零

攻擊、格擋、閃避、治療、支援與裁定都會改變下一步選擇：

- 擋下重擊後，手腕、肩膀、站位或武裝回收應受影響；
- 閃開攻擊後，角色可能被逼進不利距離、場地邊界、無影區或結界死角；
- 治療或支援會消耗時間、靈力、站位與保護窗口，可能意味著另一人被暫時放棄；
- 裁定員與醫療人員介入時，須改變場上節奏、公開觀感或下一步排程。

不得把每次攻防寫成狀態歸零的回合制互換。

## 5. 戰鬥節奏須有快慢與鏡頭

戰鬥段落應依壓力改變句子密度與鏡頭距離：

- 開戰前可寫觀察、站位、呼吸與場地；
- 第一輪交鋒應快速建立危險；
- 誤判、受傷、武裝受力或裁定介入時拉近鏡頭；
- 決勝前讓讀者知道角色正在賭什麼；
- 決勝後停在身體、排程、關係或公開觀感的後座上。

不得整場都用同一種句長與節奏，也不得用旁白總結取代交鋒過程。

## 6. 不同武裝類型必須打出差異

器物型、裝著型、生物型、群體型、場域型與複合型武裝的戰鬥感應明顯不同：

- 近戰武裝重視距離、刃線、回收、重心與空檔；
- 遠距武裝重視視線、彈道、遮蔽、連射節奏與死角；
- 絲線、棋盤、結界、場域型重視布置、節點、誘導、封路與破局；
- 治療與支援型重視站位、保護窗口、取捨與誰被暫時放棄；
- 裝著或獸化型重視身體變化、抓地、感官、衝撞與自傷風險；
- 生物型或伴生存在須區分本人判斷、伴體行動、指令與受力後果。

不得讓不同角色的戰鬥只換招式名，實際節奏與勝負邏輯完全相同。

## 7. 戰鬥中的人物性不可消失

角色在戰鬥中仍是本人。勝負選擇應反映其性格、習慣、恐懼、驕傲、偏心、責任或不想承認的弱點。

- 謹慎者不應突然無理由豪賭；
- 衝動者可以因一次吃虧被迫收斂，但不會瞬間變成完美戰術家；
- 支援者的取捨應有壓力，而不是永遠剛好救到所有人；
- 熟人對戰中的猶豫、偏心、讀錯與不願下重手，應成為可見戰鬥因素；
- 臨時對手也應有自己的打法美感、判斷失誤與敗後位置。

戰鬥不是人物暫時消失、只剩能力互撞的段落。

# 二十、能力描寫專項層

能力描寫不得只寫效果名稱、光效或結果。凡能力在正文中承擔勝負、救援、調查、關係壓力或世界推進，必須讓讀者理解其最低成立條件。能力可以神祕，但若要用來解決主衝突，讀者就必須先看懂它如何成立、如何被反制、使用者失去什麼。

## 1. 能力解決問題前，讀者必須先看懂條件

若某能力要決定勝負、救人、阻止危機或改變事件結果，正文須先自然呈現至少三項：

- 能力媒介：武裝、身體、靈力、聲音、影子、絲線、場域、血脈、名字、光紋或其他正式媒介；
- 作用對象：本人、對手、同伴、武裝、物件、區域、事件進程或殘留痕跡；
- 作用範圍：一點、一線、一人、一物、小區域或整場域；
- 成立時間：一瞬、半拍、數秒、維持狀態或觸發後延遲；
- 可被察覺之徵象：光、聲、溫度、重力、影子、紋路、疼痛、靈力流向或空氣變化；
- 可被反制之方式：離開範圍、切斷媒介、改變節奏、逼錯目標、污染靈力、破壞節點或誤導判斷；
- 使用代價：靈力、體力、精神、身體負荷、武裝穩定度、時間、站位、排程、關係或公開風險。

神祕能力可保留未知，但未知能力不得直接替角色無條件解決主衝突；若讀者尚未理解，能力更適合製造麻煩、壓力或錯判。

## 2. 每種能力都要有自己的身體語言

能力不是只換顏色的特效。不同能力應在姿勢、重心、視線、手部動作、呼吸、武裝軌跡與場地互動上形成差異。

- 時間系重視節拍、停頓、殘影、時印、半拍與判斷代價；
- 影系重視光源、死角、貼地路徑、影子連續性與無影區；
- 絲線／棋盤／結界系重視節點、布置、封路、誘導與破局；
- 重力／場域系重視腳底、膝蓋、脊背、空氣壓迫與移動線變形；
- 光紋／星紋系重視視線、反射、標記、軌跡與辨識錯誤；
- 治療／支援系重視距離、觸碰、保護窗口、取捨與暫時放棄；
- 裝著／獸化系重視抓地、骨骼、感官、衝撞、自傷與常人動作差異；
- 生物型／伴生型重視本人判斷、伴體行動、指令延遲、分工與受力後果；
- 名字／記憶／封存系重視稱呼、邊界、記錄、缺漏、回聲與資訊權限。

同一場戰鬥中，不同能力的讀感必須不同；不得只以「靈光一閃」「能力發動」替代能力差異。

## 3. 限制、弱點與代價必須進入場面

能力越強，限制越要可見。正文不得只在設定表寫限制，場面中也要讓限制影響人物選擇。

- 限制決定角色不能怎麼打；
- 弱點決定對手如何反制；
- 代價決定角色使用後失去什麼；
- 後果決定能力如何改變世界、制度、觀眾、醫療、排程或關係。

若能力使用後沒有任何身體、位置、靈力、精神、制度或關係後座，該能力不得頻繁用來解決問題。

## 4. 能力不得臨時改規則

同一能力在不同場面可因顯現層級、正式競技、本體顯現、情緒、場地與負荷而有強弱變化，但其核心媒介、作用邏輯、限制與反制不得為了單場勝負臨時改寫。

候選正文可寫出不穩定雛形、危急觸發或未完全掌握的能力表現；但若該表現將構成能力突破、新型態、長期穩定用法或角色階段提升，未經使用者正式採用與結算前不得承接。

## 5. 能力必須像使用者本人

能力描寫須反映角色性格、價值觀、習慣、恐懼、偏心、審美與武裝本質。

同樣是攻擊，有人會正面壓過去，有人會誘導，有人會先保護退路，有人會犧牲站位換取同伴窗口。能力不是角色暫時變成戰鬥機器，而是角色本人在壓力下最直接的選擇方式。

## 6. 能力也會改變日常與世界，但不能取代生活

能力可以自然影響日常、制度、醫療、交通、社團、通訊、觀戰與城市設備；但不得成為一鍵解法。

- 時間能力不能無代價縮短所有等待或抹消失敗；
- 治療能力不能抹除勝敗、公開觀感與關係裂痕；
- 記憶／名字能力不能取代調查與人際信任；
- 結界／場域能力不能讓所有危機都被安全隔開；
- 高機動能力不能取消路線、門禁、權限與生活阻力。

能力應讓世界更有層次，而不是讓生活問題消失。

# 二十一、傷勢、失敗與後座

## 1. 正式競技允許真實傷害成立

依最新版引擎，正式競技若採本體顯現，傷勢、失血、骨折、靈力失衡、武裝受損、失去續戰能力與醫療中止皆可依法成立。

正文不得因角色重要、朋友關係或校園背景，自動避開合理傷害。

## 2. 候選正文可寫當場與短期後果

AI 可自行安排：

- 當場可理解之傷勢；
- 因傷被迫認輸或遭裁定中止；
- 短期無法上場、改約、複查、換排程或被迫旁觀；
- 公開失敗造成的不甘、尷尬、逃避或關係摩擦；
- 勝者付出的負荷與後續壓力。

但死亡、長期退賽、永久失能、本源重大創傷之固定後續與篇章級重大失位，不得自行定案。

## 3. 醫療不等於抹除事件

醫療可依世界設定處理傷勢，但不能抹除：

- 已成立的勝敗；
- 已錯失的機會；
- 已暴露的打法；
- 已形成的公開觀感；
- 已留下的關係摩擦與下一步壓力。

敗者不必立刻接受結果；勝者也不必立刻輕鬆。

# 二十二、世界、能力與危機邊界

## 1. 世界生活感

本作為現代高科技都市與靈力科技並存之社會。終端、推播、公告、學院制度、社團、官方與城市設備可自然進入正文，但只能服務人物與事件，不得取代人物戲。

## 2. 異能武裝

異能武裝為使用者靈魂延伸，平時收於靈魂內，需要時由本人意志與靈力召喚、具現或展開。

正文不得：

- 把武裝寫成平日日常攜帶的普通器材；
- 自行補猜未公開型態、進化、外觀、壽命或反轉結果；
- 讓日常麻煩靠能力一鍵處理；
- 反覆以「沒有召喚武裝」之類句子進行防錯聲明。

## 3. 危機與暗線

危機濃度必須服從現行篇章骨架。

在僅允許低濃度伏筆之階段，可留下具體但有限的痕跡，例如：

- 某項設備紀錄不合常理；
- 某個人有可疑但尚不足定案的行動；
- 某份資料、物件或排程留下可追查問題。

不得提前讓危機搶走競技與人物舞台，也不得以抽象不安或萬能「異常」代替具體事件。

## 4. 原創角色與世界拓展

候選正文可依最新版完整創作引擎之 R3／R3-A 邊界，自行新增必要的新角色、臨時對手、支線配角、教師／學生／職員、社團人物、城市職業者、場地、店鋪、學院設施、普通制度、生活靈力科技、外校與城市日常細節，用以讓事件自然成立、擴大世界生活感、補足競技對手與場面因果。

新創角色須有自己的目的、尊嚴、時間、判斷、求勝理由與生活位置；不得只作為主角見證人、資訊遞送者、關係催化工具、善後工具或預定勝者的失敗裝置。正式競技中的臨時對手亦須具備可理解之武裝、打法、限制、反制與勝敗因果。

新創世界細節須服務人物行動與事件因果；可補足日常生活、競技流程、城市機能、社團運作、外校文化與低風險靈力科技，但不得以展示設定取代人物戲。

上述候選新增內容未經使用者明確正式採用並完成結算前，不得進入正式設定、正史、長線保護或後續承接；不得藉由新增角色或世界設定改寫核心世界法則、已成立人物資料、能力硬邊界、正式關係、資訊權限、長線危機真相、組織核心目的、代表隊最終結果、固定配對或不可逆重大後果。

# 二十三、生成途中之內部重寫與停止原則

若正文滑向下列失真方向，必須在內部改換場面因果後重寫：

- 本章只留下氣氛、公告、旁觀或預告，沒有支付具體變化；
- 群像人物只為同一主角、同一情緒或同一主題服務；
- 世界制度、科技或場地只展示設定，沒有製造選擇成本或後座；
- 場景沒有進場理由、離場切點或可承接變化，只是在保存流程；
- 場面只有表面事件，沒有內在拉扯或微張力；
- 承重場景缺少行動、反應與決定鏈，資訊展示後沒有改變人物下一步；
- 人物開始替作者說出本章答案；
- 關係、擔心、嫉妒、不甘或歉意被台詞／旁白講滿，缺少潛台詞；
- 視角變成全知報告，沒有明確感知焦點或資訊差；
- 承重角色沒有情緒弧，事件後情緒被整理乾淨；
- 關係靠旁人或旁白直接標註；
- 配角或對手只剩功能；
- 正式競技被匿名化、觀念化或安全化；
- 戰鬥只剩招式名稱、光效與結果，缺少距離、重心、受力、靈力流向與身體負荷；
- 能力只剩效果名稱、光效或結果，缺少媒介、作用對象、範圍、徵象、反制、代價與角色身體語言；
- 決勝依賴未成立規則、臨時能力或對手失智；
- 傷勢、失敗或摩擦被快速漂亮地整理乾淨；
- 日常收尾只是用來沖淡前段代價；
- 新場景只是前次事件換皮重播；
- 伏筆只剩抽象不安，沒有可支付對象、位置、物件或制度痕跡；
- 場景沒有波峰，結束時人物行動、資訊差、情緒位置與下一步安排都沒有改變；
- 章末只剩漂亮總結或抽象懸念，沒有具體未解狀態。
- 固定小隊連續壟斷承重，或為了公平讓全員硬出場卻沒有本人目的；
- 支線只留下「以後重要」的空泛提示，沒有下一個可見台階；
- 一章新增過多必記資訊，使角色、能力、制度、場地或伏筆擠成資料展示；
- stakes 不具體，角色輸贏、錯過或受傷後沒有明確失去之物；
- 重大事件被單場塞滿並迅速收乾淨，沒有前置壓力或後座落地；
- 重複場地或同類場面只重播上一章功能，沒有新增人物或事件變化；
- 章節開頭習慣性從流程交代開始，而不是從變動點或失常流程切入。
- 承重場景缺少「目標 → 阻力 → 結果 → 反應 → 抉擇」節拍，場景結束後沒有新決定或壓力轉向；
- 嘗試結果只有成功或失敗，沒有 Yes-but、No-and 或 No-but 所造成的後續推力；
- 內心戲替角色整理主題或關係結論，沒有貼近當下誤認、逃避、漏看或動作替代；
- 對話只在解釋資訊，沒有推動下一步、改變距離、隱瞞／暴露、製造誤會或調整張力；
- 呼吸點把壓力清空，或幽默取消傷勢、敗北、關係裂痕與事件代價；
- 章節承諾類型混亂，開了場域、問題、人物、事件多條承諾卻沒有支付主要承諾；
- 張力與釋放全章同一密度，沒有高低節拍或呼吸轉折。
- 伏筆只有可疑氣氛，沒有可觸發條件或支付方向；
- 重大事件後沒有身體、認知、行動或關係反應，角色下一步與事件前相同；
- 敘事距離失控，貼身場面變作者總結，或資訊場面故意朦朧；
- 為了滿足規則顯性化而越修越像教科書、流程報告或設定文件；
- 章末鉤子空降新危機，或用抽象不安硬拉期待；
- 人物弧線每章都被整理成成熟、理解或和解，失去活人反覆；
- 摘要段落偷渡承重事件，或只是無方向流水帳。

- 戀愛或日常只剩甜場、漂亮互動或資訊整理，沒有生活阻力與選擇成本；
- 情緒被旁白或旁人直接標註，沒有落在動作、停頓、物件或不成熟反應上；
- 角色聲線只靠口癖、句尾或固定短句區分，遮掉名字後分不出人物；
- 對話只靠台詞推進，逃避、擔心、偏心、不甘或靠近沒有落到物件、站位、停頓或動作；
- 轉場頻繁但沒有新壓力、新資訊差、新後座或人物下一步變化；
- 章節開頭預設全知報告，同時交代時間、地點、制度、多人心理與本章主題；
- 重複意象只因漂亮或熟悉而出現，沒有功能變化；
- 朗讀時像報告、口號、設定摘要或漂亮句堆疊；
- 承重事件被摘要帶過，低承重流程卻被完整展開。

- 段落先總結心理、再補外部刺激，導致讀者無法跟著角色經驗場面；
- 感官描寫與視角人物無關，或為了豐富而堆疊五感；
- 承重角色整章只旁觀、理解、被照顧或被推動，沒有主動行動改變場面；
- 對手或反派沒有主動反制，只作為等待主角展示能力的障礙物；
- 低顯著流程佔用完整場景篇幅，真正轉折反而被摘要帶過；
- 章節中段只重複開頭已知壓力，沒有策略、限制、誤判、資訊差或支線台階推進；
- 同一段落塞入過多動作、能力說明、內心分析、世界設定與伏筆，造成焦點過載。

重寫原則:

1. 先保住人物真正要做的事，不保住漂亮句子。
2. 先保住合法因果與真實代價，不保住預定效果。
3. 單場具名勝負、當場傷勢與短期後座可依法改寫重建，不必因害怕越權而抽空事件。
4. 若必須先決定受保護之長線核心或不可逆重大事項，才停止並指出待決定項目。

# 二十五、與完稿後驗稿卡之接口

候選正文完成後，本卡任務結束。正式採用前，另依本輪指定之《完稿後驗稿卡》驗收完整候選正文。

驗稿時應特別確認：

- 本章主要事件是否真的由具名人物承擔；
- 本章是否有承諾、推進與具體支付，而非只留下氣氛或預告；
- 群像是否存在不同生活軌道與選擇成本，而非集體服務同一情緒；
- 場景是否有進場理由、場景波峰、離場切點與可承接後座；
- 承重場景是否具備「目標 → 阻力 → 結果 → 反應 → 抉擇」節拍，並讓角色下一步改變；
- Try／Fail 結果是否形成 Yes-but、No-and 或 No-but，而非單純成功／失敗；
- 內心戲是否貼近視角人物當下可承受、可誤認或不願承認的內容，而非作者總結；
- 承重對話是否推動下一步、改變距離、隱瞞／暴露資訊、製造誤會或調整張力；
- 呼吸點是否保留事件代價，幽默是否來自人物與場面而非硬插梗；
- 本章主要承諾類型是否清楚，章末是否支付或轉化該承諾；

- 補強 stakes 具體化，要求每章明確角色可能失去的名額、時間、面子、觀眾評價、信任、主導權、下一場準備或自我認知。
- 補強重大事件之場景序列，要求高承重事件形成「前置壓力 → 當場支付 → 後座落地」之連鎖，可同章或跨章支付。
- 補強同類場景防膩與章節開頭變動點，避免公告牆、觀戰區、醫療門外、社團桌等場面重播同一功能，並避免每章從流程交代開始。
- 同步更新正文候選短板機、生成途中重寫條件與完稿後驗稿接口。

## v2.0｜伏筆資訊・場景波峰・修稿分層全面校正

- 新增「伏筆資訊、場景波峰與長篇修稿層」，將長篇連載穩定性納入起稿伴隨規則。
- 補強伏筆支付距離，區分近程、中程、長程伏筆，避免暗線只剩抽象不安或重複氣氛。
- 補強場景波峰，要求承重場景推到人物姿態、計畫、隱瞞、距離或判斷無法完全維持的位置，並揭露新資訊或留下具體後座。
- 補強「想要／現實阻力」張力線，使日常、戀愛、社團、觀戰、醫療與正式競技皆能由具體阻力推動。
- 補強資訊三層，區分讀者知道、角色知道與角色以為自己知道，並要求資訊差推動行動。
- 補強情感場地與章末具體未解狀態，避免場地歸零、章末漂亮總結或抽象懸念。
- 新增分層修稿順序：結構、場景、人物、資訊、能力／戰鬥、語言依序檢查，禁止先修漂亮句子而保留壞因果。
- 同步更新正文候選短板機、生成途中重寫條件與完稿後驗稿接口。

## v1.9｜潛台詞・視角焦點・情緒弧與微張力補強

- 新增「潛台詞、視角焦點與讀者感受層」，要求每場承重戲同時具備表面事件與內在拉扯，避免正文只剩流程、抒情或資訊展示。
- 補強微張力原則：話語與真意、行動與宣稱、想幫忙與成本、想逃避與被留下等小矛盾須持續推動場面。
- 補強行動—反應—決定鏈，要求承重場景在事件後改變至少一名角色下一步，不得只展示資訊。
- 補強潛台詞與非直白情緒，關係、擔心、嫉妒、不甘與歉意須優先藏在技術話題、時間、物件、停頓、刪改訊息與多做一步的行動裡。
- 補強視角焦點與資訊差，禁止全知報告式平均敘述；切換視角必須帶來新的資訊差、情緒差或判斷差。
- 補強情緒弧與句子節奏功能，要求承重角色在章內有開場、中段受壓與章末殘留狀態，並依場面功能調整句子密度。
- 同步更新正文候選短板機、生成途中重寫條件與完稿後驗稿接口。

## v1.8｜場景承重・群像調度・章節支付全面補強

- 新增「章節承重、群像調度與剪輯支付層」，要求每章具備承諾、推進與具體支付，不得只靠公告、旁觀、氣氛或預告成立。
- 補強群像生活軌道調度，要求多名承重人物帶著各自目的、時間壓力與不願失去之物進場，並讓生活軌道互相干擾。
- 補強對手與反派壓力設計，要求承重對手同時提供外部阻力與內部對照，反派壓力須落為可感、可追查的具體行動或後果。
- 補強世界摩擦與制度後座，要求門禁、醫療、觀戰、社團、靈軌、通訊、公開紀錄等世界細節製造選擇成本與資訊差。
- 補強章節剪輯與切場規則，允許跳接與略過低承重流程，但每個場景必須具備進場理由、離場切點與可承接變化。
- 新增完稿前五問，作為起稿卡內部退稿與重寫依據，避免章節只保留漂亮句子而沒有故事位置改變。
- 同步更新正文候選短板機、生成途中重寫條件與完稿後驗稿接口。

## v1.7｜能力描寫專項層補強

- 新增「能力描寫專項層」，要求能力不只呈現效果，也要具備媒介、作用對象、作用範圍、成立時間、可見徵象、反制方式與使用代價。
- 補強能力身體語言：時間、影、絲線／棋盤／結界、重力／場域、光紋／星紋、治療／支援、裝著／獸化、生物／伴生、名字／記憶／封存等能力須呈現不同姿勢、重心、視線、手部動作、呼吸與場地互動。
- 補強限制、弱點與代價必須進入場面，不得只留在設定表；能力越強，越需要讓限制與反制影響人物選擇。
- 補強能力不得臨時改規則：候選可寫不穩定雛形或危急觸發，但能力突破、新型態、長期穩定用法與階段提升仍須正式採用與結算後才可承接。
- 補強能力與角色本人、日常制度及世界生活之連動；能力應讓世界更有層次，不得成為一鍵解法。
- 同步更新正文候選短板機、生成途中重寫條件與完稿後驗稿接口。

## v1.6｜戀愛與日常場面描寫層補強

- 新增「戀愛與日常場面描寫層」，要求日常不只是休息場、甜場或資訊整理場，而必須保留人物各自的生活軌道與事件後座。
- 補強慢熱戀愛之「選擇成本」原則：靠近須由等待、改安排、承擔不便、保留話語、幫錯忙或不成熟反應成立，不得靠旁白或旁人直接標註。
- 補強情緒非語言呈現：視線、手勢、腳步、聲音、停頓、訊息刪改、座位距離與物件位置優先於「在意／心動／變近」等總結句。
- 補強戰鬥、傷勢、正式選拔、醫療規則、社團登記、通訊與城市生活如何參與關係變化；日常不得沖淡戰鬥代價，受傷照顧不得自動成為關係答案。
- 同步更新正文候選短板機與生成途中重寫條件，避免戀愛與日常滑成甜場整理、漂亮互動或資訊補充。

## v1.5｜戰鬥場面描寫層補強

- 新增「戰鬥場面描寫層」，要求戰鬥不只支付合法勝負，也要呈現目標、進度、距離、重心、受力、靈力流向、能力成立順序、身體負荷與攻防後狀態改變。
- 補強戰鬥節奏與鏡頭規則：開戰前可慢、交鋒時加速、誤判與受傷時拉近、決勝後停在身體／排程／關係／公開觀感後座。
- 補強不同武裝類型之描寫差異，避免角色只換招式名但戰鬥節奏與勝負邏輯相同。
- 補強戰鬥中的人物本人性，要求角色性格、習慣、恐懼、偏心、責任與臨時對手的判斷失誤都能進入勝負因果。
- 同步更新正文候選短板機與內部重寫條件，要求正式戰鬥不可只剩招式名稱、光效與結果。
